# Claude Engineer

Claude Engineer is an interactive command-line interface (CLI) that uses the power of Anthropic's Claude-3.5-Sonnet model to assist with software development tasks.This framework enables Claude to generate and manage its own tools, continuously expanding its capabilities through conversation. Available both as a CLI and a modern web interface ## Commands

- `help` - Help
- `run` - Run
- `info` - Info
- `status` - Status

## Features

- Core functionality from Doriandarko/ai-engineer

## Usage

Run any command: `ai-engineer <command> [args]`
