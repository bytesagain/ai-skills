# api

Test and debug REST APIs with request building and response parsing. Use when sending HTTP requests, testing endpoints, or inspecting API responses.

## Commands

### `get`

```bash
scripts/script.sh get <url>
```

### `post`

```bash
scripts/script.sh post <url data>
```

### `headers`

```bash
scripts/script.sh headers <url>
```

### `status`

```bash
scripts/script.sh status <url>
```

### `time`

```bash
scripts/script.sh time <url>
```

### `json`

```bash
scripts/script.sh json <url>
```

## Requirements

- bash 4.0+
- curl
- python3
