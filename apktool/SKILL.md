# Apktool

A tool for reverse engineering Android apk files ## Commands

- `help` - Help
- `run` - Run
- `info` - Info
- `status` - Status

## Features

- Core functionality from iBotPeaches/Apktool

## Usage

Run any command: `apktool <command> [args]`

## Examples

```bash
apktool help
apktool run
```

## When to Use

- when you need quick apktool from the command line
- to automate apktool tasks in your workflow

## Output

Returns formatted output to stdout. Redirect to a file with `apktool run > output.txt`.


## Features

- Simple command-line interface for quick access
- Local data storage with JSON/CSV export
- History tracking and activity logs
- Search across all entries
- No external dependencies required

## Quick Start

```bash
# Check status
apktool status

# View help
apktool help

# View statistics
apktool stats

# Export data
apktool export json
```

## How It Works

Apktool stores all data locally in `~/.local/share/apktool/`. Each command logs activity with timestamps for full traceability.
