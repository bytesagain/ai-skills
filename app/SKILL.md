# app

Scaffold, build, and manage application projects locally. Use when initializing projects, checking dependencies, or packaging apps.

## Commands

### `init`

```bash
scripts/script.sh init <name>
```

### `tree`

```bash
scripts/script.sh tree <dir>
```

### `size`

```bash
scripts/script.sh size <dir>
```

### `deps`

```bash
scripts/script.sh deps <dir>
```

### `clean`

```bash
scripts/script.sh clean <dir>
```

### `lines`

```bash
scripts/script.sh lines <dir>
```

## Requirements

- bash 4.0+
- git
