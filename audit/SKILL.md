# audit

Audit system security with user, port, and permission checks. Use when reviewing security posture, finding open ports, or checking file permissions.

## Commands

### `users`

```bash
scripts/script.sh users
```

### `ports`

```bash
scripts/script.sh ports
```

### `perms`

```bash
scripts/script.sh perms <dir>
```

### `ssh`

```bash
scripts/script.sh ssh
```

### `firewall`

```bash
scripts/script.sh firewall
```

### `packages`

```bash
scripts/script.sh packages
```

## Requirements

- bash 4.0+

## Data Storage

Data stored in `~/.local/share/audit/`.
