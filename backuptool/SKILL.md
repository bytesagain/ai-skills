# BackupTool

File and directory backup utility. Create timestamped backups, compress with gzip, manage backup rotation, restore from backups, and schedule automatic backups.

## Quick Start

Run `backuptool help` for available commands and usage examples.

## Features

- Fast and lightweight — pure bash with embedded Python
- No external dependencies required
- Works on Linux and macOS

## Usage

```bash
backuptool help
```
