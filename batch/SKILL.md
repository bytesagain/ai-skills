# batch

Process files in bulk with rename, convert, and compress operations. Use when batch renaming, resizing, or converting file formats.

## Commands

### `rename`

```bash
scripts/script.sh rename <dir from to>
```

### `compress`

```bash
scripts/script.sh compress <dir>
```

### `count`

```bash
scripts/script.sh count <dir ext>
```

### `list`

```bash
scripts/script.sh list <dir>
```

### `delete-empty`

```bash
scripts/script.sh delete-empty <dir>
```

### `total-size`

```bash
scripts/script.sh total-size <dir>
```

## Requirements

- bash 4.0+

## Data Storage

Data stored in `~/.local/share/batch/`.
