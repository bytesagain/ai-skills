# build

Compile, build, and package projects with multi-language support. Use when compiling code, running builds, or creating release artifacts.

## Commands

### `detect`

```bash
scripts/script.sh detect <dir>
```

### `make`

```bash
scripts/script.sh make <dir>
```

### `clean`

```bash
scripts/script.sh clean <dir>
```

### `deps`

```bash
scripts/script.sh deps <dir>
```

### `size`

```bash
scripts/script.sh size <dir>
```

### `artifacts`

```bash
scripts/script.sh artifacts <dir>
```

## Requirements

- bash 4.0+
