# Webpackmonitor

A tool for monitoring webpack optimization metrics through the development process ## Commands

- `help` - Help
- `run` - Run
- `info` - Info
- `status` - Status

## Features

- Core functionality from bundle-analyzer/bundle-analyzer

## Usage

Run any command: `bundle-analyzer <command> [args]`

## When to Use

- when you need quick bundle analyzer from the command line
- to automate bundle tasks in your workflow

## Output

Returns results to stdout. Redirect to a file with `bundle-analyzer run > output.txt`.
