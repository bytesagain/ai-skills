# Carbon Capture — Carbon Capture Technology Reference

Quick-reference skill for carbon capture, storage, and utilization technologies.

## When to Use

- Evaluating carbon capture approaches for industrial emitters
- Comparing post-combustion, pre-combustion, and oxy-fuel methods
- Understanding Direct Air Capture (DAC) technologies
- Assessing geological CO₂ storage options
- Analyzing CCUS economics and policy incentives

## Commands

### `intro`

```bash
scripts/script.sh intro
```

Overview of carbon capture — the CCS chain, scale, and global status.

### `postcombustion`

```bash
scripts/script.sh postcombustion
```

Post-combustion capture: amine scrubbing, advanced solvents, membranes, solid sorbents.

### `precombustion`

```bash
scripts/script.sh precombustion
```

Pre-combustion and oxy-fuel combustion capture methods.

### `dac`

```bash
scripts/script.sh dac
```

Direct Air Capture — liquid solvent (Carbon Engineering) and solid sorbent (Climeworks) approaches.

### `transport`

```bash
scripts/script.sh transport
```

CO₂ transport: pipeline, ship, truck — phase diagrams and design considerations.

### `storage`

```bash
scripts/script.sh storage
```

Geological storage: saline aquifers, depleted reservoirs, basalt mineralization, monitoring.

### `utilization`

```bash
scripts/script.sh utilization
```

CO₂ utilization pathways: EOR, building materials, fuels, chemicals, mineralization.

### `economics`

```bash
scripts/script.sh economics
```

CCS economics: cost per ton CO₂, 45Q tax credits, ETS pricing, project economics.

### `help`

```bash
scripts/script.sh help
```

### `version`

```bash
scripts/script.sh version
```
