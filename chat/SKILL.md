# chat

Local chat logger and message template manager. Use when drafting messages, logging chats, or managing templates.

## Commands

### `send`

```bash
scripts/script.sh send <channel message>
```

### `read`

```bash
scripts/script.sh read <channel>
```

### `channels`

```bash
scripts/script.sh channels
```

### `search`

```bash
scripts/script.sh search <query>
```

### `export`

```bash
scripts/script.sh export <channel file>
```

### `clear`

```bash
scripts/script.sh clear <channel>
```

## Data Storage

Data stored in `~/.local/share/chat/`.
