# Chat2Db

AI-driven database tool and SQL client, The hottest GUI client, supporting MySQL, Oracle, PostgreSQL, DB2, SQL Server, DB2, SQLite, H2, ClickHouse, and more.

Inspired by [CodePhiliaX/Chat2DB]([configured-endpoint]) (25,224+ GitHub stars).

## Commands

- `help` - Help
- `run` - Run
- `info` - Info
- `status` - Status

## Features

- Core functionality from CodePhiliaX/Chat2DB

## Usage

Run any command: `chat2db <command> [args]`
