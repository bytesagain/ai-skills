# check

Run health checks on URLs, files, disk, and DNS. Use when verifying uptime or validating configs.

## Commands

### `url`

```bash
scripts/script.sh url <url>
```

### `disk`

```bash
scripts/script.sh disk
```

### `dns`

```bash
scripts/script.sh dns <domain>
```

### `port`

```bash
scripts/script.sh port <host port>
```

### `file`

```bash
scripts/script.sh file <path>
```

### `memory`

```bash
scripts/script.sh memory
```

## Data Storage

Data stored in `~/.local/share/check/`.
