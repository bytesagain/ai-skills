# chinese-calendar-cn

Chinese calendar with zodiac, festivals, and solar terms (中文). Use when checking lunar dates.

## Commands

### `today`

```bash
scripts/script.sh today
```

### `zodiac`

```bash
scripts/script.sh zodiac <year>
```

### `festival`

```bash
scripts/script.sh festival <year>
```

### `solar-term`

```bash
scripts/script.sh solar-term <month>
```

### `convert`

```bash
scripts/script.sh convert <date>
```

### `list`

```bash
scripts/script.sh list
```

## Data Storage

Data stored in `~/.local/share/chinese-calendar-cn/`.
