# Chromedeveditor

Chrome Dev Editor is a developer tool for building apps on the Chrome platform - Chrome Apps and Web Apps, in JavaScript or Dart. (NO LONGER IN ACTIVE DEVELOPMENT)

Inspired by [googlearchive/chromedeveditor]([configured-endpoint]) (2,919+ GitHub stars).

## Commands

- `help` - Help
- `run` - Run
- `info` - Info
- `status` - Status

## Features

- Core functionality from googlearchive/chromedeveditor

## Usage

Run any command: `chromedeveditor <command> [args]`

## Examples

```bash
chromedeveditor help
chromedeveditor run
```

## When to Use

- as part of a larger automation pipeline
- when you need quick chromedeveditor from the command line

## Output

Returns reports to stdout. Redirect to a file with `chromedeveditor run > output.txt`.

## Configuration

Set `CHROMEDEVEDITOR_DIR` environment variable to change the data directory. Default: `~/.local/share/chromedeveditor/`
