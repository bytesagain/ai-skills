# Claude Code Templates

CLI tool for configuring and monitoring Claude Code

Inspired by [davila7/claude-code-templates]([configured-endpoint]) (22,907+ GitHub stars).

## Commands

- `help` - Help
- `run` - Run
- `info` - Info
- `status` - Status

## Features

- Core functionality from davila7/claude-code-templates

## Usage

Run any command: `claude-code-templates <command> [args]`

## Examples

```bash
claude-code-templates help
claude-code-templates run
```

## When to Use

- as part of a larger automation pipeline
- when you need quick claude code templates from the command line

## Output

Returns structured data to stdout. Redirect to a file with `claude-code-templates run > output.txt`.

## Configuration

Set `CLAUDE_CODE_TEMPLATES_DIR` environment variable to change the data directory. Default: `~/.local/share/claude-code-templates/`
