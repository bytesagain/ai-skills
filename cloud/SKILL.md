# cloud

Check cloud service endpoints, regions, and costs. Use when testing APIs or estimating costs.

## Commands

### `regions`

```bash
scripts/script.sh regions <provider>
```

### `cost`

```bash
scripts/script.sh cost <hours rate>
```

### `ips`

```bash
scripts/script.sh ips
```

### `dns`

```bash
scripts/script.sh dns <domain>
```

### `ping`

```bash
scripts/script.sh ping <host>
```

### `status`

```bash
scripts/script.sh status <service>
```

## Data Storage

Data stored in `~/.local/share/cloud/`.
