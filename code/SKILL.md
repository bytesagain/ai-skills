# code

Analyze source code for complexity, quality, and structure. Use when counting lines, finding TODOs, or measuring code quality.

## Commands

### `count`

```bash
scripts/script.sh count <dir>
```

### `todo`

```bash
scripts/script.sh todo <dir>
```

### `functions`

```bash
scripts/script.sh functions <file>
```

### `imports`

```bash
scripts/script.sh imports <file>
```

### `complexity`

```bash
scripts/script.sh complexity <file>
```

### `dupes`

```bash
scripts/script.sh dupes <dir>
```

## Requirements

- bash 4.0+
