# connect

Test network connectivity to hosts and ports. Use when checking reachability or measuring latency.

## Commands

### `tcp`

```bash
scripts/script.sh tcp <host port>
```

### `ping`

```bash
scripts/script.sh ping <host>
```

### `dns`

```bash
scripts/script.sh dns <domain>
```

### `trace`

```bash
scripts/script.sh trace <host>
```

### `http`

```bash
scripts/script.sh http <url>
```

### `scan`

```bash
scripts/script.sh scan <host>
```

## Data Storage

Data stored in `~/.local/share/connect/`.
