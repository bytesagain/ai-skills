# container

Manage Docker containers with status, logs, and cleanup. Requires docker.

## Commands

### `list`

```bash
scripts/script.sh list
```

### `stats`

```bash
scripts/script.sh stats
```

### `logs`

```bash
scripts/script.sh logs <name>
```

### `inspect`

```bash
scripts/script.sh inspect <name>
```

### `cleanup`

```bash
scripts/script.sh cleanup
```

### `images`

```bash
scripts/script.sh images
```

## Data Storage

Data stored in `~/.local/share/container/`.
