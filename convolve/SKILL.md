# Convolve

Compute convolve operations. Use when you need to solve convolve equations, run statistical analysis, or simulate numerical models.

## When to Use

- **status**: Show current status
- **add**: Add new entry
- **list**: List all entries
- **search**: Search entries
- **calculate**: Evaluate math expression
- **remove**: Remove entry by number
- **export**: Export data to file
- **stats**: Show statistics
- **config**: View or set config

## Commands

### `status`

```bash
scripts/script.sh status
```
