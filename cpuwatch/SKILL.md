# CPUWatch

CPU usage monitor and process analyzer. Real-time CPU utilization tracking, per-core usage breakdown, top CPU-consuming processes, load average trends, and temperature monitoring where available. Keep your CPU usage in check.

## Quick Start

Run `cpuwatch help` for detailed usage information and available commands.
