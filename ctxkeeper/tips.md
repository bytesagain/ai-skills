# Context Manager — Tips

1. **Summarize early, summarize often** — Don't wait until you hit token limits. Proactive compression preserves more meaning.

2. **Prioritize by task relevance** — Context from 20 messages ago about a different topic can usually be archived safely.

3. **Archive, don't delete** — Always archive old context instead of discarding. You might need it later.

4. **Monitor your budget** — Run `budget` periodically to stay aware of token usage trends.

5. **Split by topic** — When conversations cover multiple subjects, split into topic-specific chunks for cleaner management.

6. **Merge carefully** — When combining contexts, review the merged result for contradictions or stale information.

7. **Optimize for the current task** — Context relevant to the active task should always be prioritized highest.

8. **Use restore strategically** — Only restore archived context when you actually need it, not preemptively.
