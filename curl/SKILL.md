# Curl

A command line tool and library for transferring data with URL syntax, supporting DICT, FILE, FTP, FTPS, GOPHER, GOPHERS, HTTP, HTTPS, IMAP, IMAPS, LDAP, LDAPS, MQTT, MQTTS, POP3, POP3S, RTMP, RTMPS, RTSP, SCP, SFTP, SMB, SMBS, SMTP, SMTPS, TELNET, TFTP, WS and WSS. libcurl offers a myriad of powerful features

Inspired by [curl/curl]([configured-endpoint]) (41,097+ GitHub stars).

## Commands

- `help` - Help
- `run` - Run
- `info` - Info
- `status` - Status

## Features

- Core functionality from curl/curl

## Usage

Run any command: `curl <command> [args]`

## Output

Returns structured data to stdout. Redirect to a file with `curl run > output.txt`.

## Configuration

Set `CURL_DIR` environment variable to change the data directory. Default: `~/.local/share/curl/`
