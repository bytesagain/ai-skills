# Dataease

🔥 人人可用的开源 BI 工具，数据可视化神器。An open-source BI tool alternative to Tableau. ## Commands

- `help` - Help
- `run` - Run
- `info` - Info
- `status` - Status

## Features

- Core functionality from data-dashboard/data-dashboard

## Usage

Run any command: `data-dashboard <command> [args]`

## Examples

```bash
data-dashboard help

```

## When to Use

- when you need quick data dashboard from the command line
- to automate data tasks in your workflow

## Output

Returns summaries to stdout. Redirect to a file with `data-dashboard run > output.txt`.
