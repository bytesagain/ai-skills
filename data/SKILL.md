# data

Transform, validate, and analyze data files in CSV, JSON, and text. Use when cleaning datasets, converting formats, or computing statistics.

## Commands

### `csv-info`

```bash
scripts/script.sh csv-info <file>
```

### `json-keys`

```bash
scripts/script.sh json-keys <file>
```

### `stats`

```bash
scripts/script.sh stats <file>
```

### `sample`

```bash
scripts/script.sh sample <file n>
```

### `unique`

```bash
scripts/script.sh unique <file col>
```

### `sort-csv`

```bash
scripts/script.sh sort-csv <file col>
```

## Requirements

- bash 4.0+
- python3
