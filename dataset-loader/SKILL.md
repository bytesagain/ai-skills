# Datasets

🤗 The largest hub of ready-to-use dataset-loader for AI models with fast, easy-to-use and efficient data manipulation tools ## Commands

- `help` - Help
- `run` - Run
- `info` - Info
- `status` - Status

## Features

- Core functionality from huggingface/dataset-loader

## Usage

Run any command: `dataset-loader <command> [args]`

## Examples

```bash
dataset-loader help
dataset-loader run
```

## When to Use

- when you need quick dataset loader from the command line
- to automate dataset tasks in your workflow

## Output

Returns results to stdout. Redirect to a file with `dataset-loader run > output.txt`.
