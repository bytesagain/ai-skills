# Deer Flow

An open-source SuperAgent use that researches, codes, and creates. With the help of sandboxes, memories, tools, skills and subagents, it handles different levels of tasks that could take minutes to hours.

Inspired by [bytedance/deer-flow]([configured-endpoint]) (30,440+ GitHub stars).

## Commands

- `help` - Help
- `run` - Run
- `info` - Info
- `status` - Status

## Features

- Core functionality from bytedance/deer-flow

## Usage

Run any command: `deer-flow <command> [args]`
