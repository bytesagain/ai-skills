# Desalination — Desalination Technology Reference

Quick-reference skill for desalination processes, membrane technology, and plant design.

## When to Use

- Comparing reverse osmosis vs thermal desalination
- Sizing and designing RO membrane systems
- Evaluating energy recovery devices
- Understanding brine management and environmental impact
- Calculating water production costs

## Commands

### `intro`

```bash
scripts/script.sh intro
```

Overview of desalination — global context, source water types, and technology landscape.

### `ro`

```bash
scripts/script.sh ro
```

Reverse osmosis fundamentals — membranes, pressure, flux, salt rejection.

### `thermal`

```bash
scripts/script.sh thermal
```

Thermal desalination: Multi-Stage Flash (MSF), Multi-Effect Distillation (MED), MVC.

### `pretreatment`

```bash
scripts/script.sh pretreatment
```

Feed water pretreatment — intake, filtration, chemical dosing, SDI/MFI testing.

### `energy`

```bash
scripts/script.sh energy
```

Energy consumption and recovery — PX, turbochargers, specific energy comparison.

### `membranes`

```bash
scripts/script.sh membranes
```

RO membrane types, manufacturers, fouling, cleaning, and replacement.

### `brine`

```bash
scripts/script.sh brine
```

Brine management — disposal methods, zero liquid discharge, environmental impact.

### `economics`

```bash
scripts/script.sh economics
```

Desalination economics — CAPEX, OPEX, LCOW, and cost reduction trends.

### `help`

```bash
scripts/script.sh help
```

### `version`

```bash
scripts/script.sh version
```
