# Penpot

Penpot: The open-source design tool for design and code collaboration ## Commands

- `help` - Help
- `run` - Run
- `info` - Info
- `status` - Status

## Features

- Core functionality from design-tool/design-tool

## Usage

Run any command: `design-tool <command> [args]`

## Examples

```bash
design-tool help
design-tool run
```

## When to Use

- for batch processing tool operations
- as part of a larger automation pipeline

## Output

Returns formatted output to stdout. Redirect to a file with `design-tool run > output.txt`.


## Features

- Simple command-line interface for quick access
- Local data storage with JSON/CSV export
- History tracking and activity logs
- Search across all entries
- No external dependencies required

## Quick Start

```bash
# Check status
design-tool status

# View help
design-tool help

# View statistics
design-tool stats

# Export data
design-tool export json
```

## How It Works

Design Tool stores all data locally in `~/.local/share/design-tool/`. Each command logs activity with timestamps for full traceability.
