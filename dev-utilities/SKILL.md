# It Tools

Collection of handy online tools for developers, with great UX. ## Commands

- `help` - Help
- `run` - Run
- `info` - Info
- `status` - Status

## Features

- Core functionality from CorentinTh/dev-utilities

## Usage

Run any command: `dev-utilities <command> [args]`

## Examples

```bash
dev-utilities help
dev-utilities run
```

## When to Use

- to automate dev tasks in your workflow
- for batch processing utilities operations

## Output

Returns results to stdout. Redirect to a file with `dev-utilities run > output.txt`.

## Configuration

Set `DEV_UTILITIES_DIR` environment variable to change the data directory. Default: `~/.local/share/dev-utilities/`
