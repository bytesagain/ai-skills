# 90Daysofdevops

This repository started out as a learning in public project for myself and has now become a structured learning map for many in the community. We have 3 years under our belt covering all things DevOps, including Principles, Processes, Tooling and Use Cases surrounding this vast topic. ## Commands

- `help` - Help
- `run` - Run
- `info` - Info
- `status` - Status

## Features

- Core functionality from MichaelCade/90DaysOfDevOps

## Usage

Run any command: `devops-journey <command> [args]`

```bash
devops-journey help
devops-journey run
```

## When to Use

- to automate devops tasks in your workflow
- for batch processing journey operations

## Output

Returns results to stdout. Redirect to a file with `devops-journey run > output.txt`.

## Configuration

Set `DEVOPS_JOURNEY_DIR` environment variable to change the data directory. Default: `~/.local/share/devops-journey/`
