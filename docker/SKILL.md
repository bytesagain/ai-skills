# docker

Manage Docker containers, images, and volumes. Requires docker CLI.

## Commands

### `ps`

```bash
scripts/script.sh ps
```

### `images`

```bash
scripts/script.sh images
```

### `volumes`

```bash
scripts/script.sh volumes
```

### `logs`

```bash
scripts/script.sh logs <container>
```

### `prune`

```bash
scripts/script.sh prune
```

### `stats`

```bash
scripts/script.sh stats
```

## Data Storage

Data stored in `~/.local/share/docker/`.
