# domain

Check domain WHOIS, DNS, and expiry info. Use when researching domains or auditing DNS.

## Commands

### `whois`

```bash
scripts/script.sh whois <domain>
```

### `dns`

```bash
scripts/script.sh dns <domain>
```

### `mx`

```bash
scripts/script.sh mx <domain>
```

### `expiry`

```bash
scripts/script.sh expiry <domain>
```

### `registrar`

```bash
scripts/script.sh registrar <domain>
```

### `ns`

```bash
scripts/script.sh ns <domain>
```

## Data Storage

Data stored in `~/.local/share/domain/`.
