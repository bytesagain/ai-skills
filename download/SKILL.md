# download

Download files with resume and batch support. Use when fetching URLs or bulk downloading.

## Commands

### `url`

```bash
scripts/script.sh url <url output>
```

### `resume`

```bash
scripts/script.sh resume <url>
```

### `batch`

```bash
scripts/script.sh batch <file>
```

### `info`

```bash
scripts/script.sh info <url>
```

### `mirror`

```bash
scripts/script.sh mirror <url>
```

### `verify`

```bash
scripts/script.sh verify <file url>
```

## Data Storage

Data stored in `~/.local/share/download/`.
