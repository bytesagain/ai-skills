# Airdrop Hunter

Track upcoming crypto airdrops and maximize your eligibility.

## Quick Start

```bash
bash scripts/airdrop.sh list              # List active airdrops
bash scripts/airdrop.sh check <wallet>    # Check wallet eligibility
bash scripts/airdrop.sh calendar          # Upcoming airdrop calendar
bash scripts/airdrop.sh strategy          # Airdrop farming strategies
```

## What It Does

| Command | Purpose |
|---------|---------|
| `list` | Active and upcoming airdrops |
| `check` | Wallet eligibility analysis |
| `calendar` | Deadline calendar |
| `strategy` | Farming strategy guide |
| `track` | Add/remove tracked airdrops |
| `history` | Past airdrop values |

## Airdrop Categories

- **Retroactive**: Reward past protocol users
- **Testnet**: Participate in testnets for allocation
- **Social**: Follow/RT/join Discord for tokens
- **Holder**: Hold specific NFTs or tokens
- **LP**: Provide liquidity for allocation

## Pro Tips

1. Use multiple wallets for different protocols
2. Always interact with testnets early
3. Bridge assets across chains for eligibility
4. Join Discord servers and get active roles
5. Track deadlines — most airdrops have snapshots
