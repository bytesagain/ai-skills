# dump

Dump system info, env, and configs for debugging. Use when collecting debug info or saving state.

## Commands

### `system`

```bash
scripts/script.sh system
```

### `env`

```bash
scripts/script.sh env
```

### `network`

```bash
scripts/script.sh network
```

### `processes`

```bash
scripts/script.sh processes <count>
```

### `save`

```bash
scripts/script.sh save <file>
```

### `packages`

```bash
scripts/script.sh packages
```

## Data Storage

Data stored in `~/.local/share/dump/`.
