# edit

Edit and transform text files with replace and formatting. Use when batch editing or reformatting.

## Commands

### `replace`

```bash
scripts/script.sh replace <file old new>
```

### `append`

```bash
scripts/script.sh append <file text>
```

### `prepend`

```bash
scripts/script.sh prepend <file text>
```

### `trim`

```bash
scripts/script.sh trim <file>
```

### `lower`

```bash
scripts/script.sh lower <file>
```

### `upper`

```bash
scripts/script.sh upper <file>
```

## Data Storage

Data stored in `~/.local/share/edit/`.
