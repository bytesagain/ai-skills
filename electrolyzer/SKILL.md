# Electrolyzer — Electrolyzer Technology Reference

Quick-reference skill for water electrolysis technologies and green hydrogen production.

## When to Use

- Comparing PEM vs alkaline vs SOEC electrolyzers
- Sizing electrolyzer systems for hydrogen production
- Evaluating electrolyzer efficiency and degradation
- Understanding stack design and balance of plant
- Analyzing green hydrogen project economics

## Commands

### `intro`

```bash
scripts/script.sh intro
```

Overview of water electrolysis — the hydrogen color spectrum and why electrolyzers matter.

### `alkaline`

```bash
scripts/script.sh alkaline
```

Alkaline electrolysis — mature technology, KOH electrolyte, nickel electrodes.

### `pem`

```bash
scripts/script.sh pem
```

PEM electrolysis — proton exchange membrane, iridium/platinum catalysts, fast response.

### `soec`

```bash
scripts/script.sh soec
```

Solid Oxide Electrolysis — high temperature, steam electrolysis, highest efficiency.

### `stackdesign`

```bash
scripts/script.sh stackdesign
```

Stack design — cell components, bipolar plates, current density, voltage curves.

### `bop`

```bash
scripts/script.sh bop
```

Balance of Plant — water purification, power supply, gas processing, cooling.

### `efficiency`

```bash
scripts/script.sh efficiency
```

Efficiency metrics — voltage, current, system efficiency, degradation rates.

### `economics`

```bash
scripts/script.sh economics
```

Green hydrogen economics — LCOH, CAPEX trends, and cost reduction pathways.

### `help`

```bash
scripts/script.sh help
```

### `version`

```bash
scripts/script.sh version
```
