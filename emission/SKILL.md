# Emission — GHG Emission Reference

Quick-reference skill for greenhouse gas emissions, carbon accounting, and climate reporting.

## When to Use

- Calculating organizational or product carbon footprints
- Understanding Scope 1, 2, 3 emission categories
- Selecting emission factors for energy, transport, materials
- Reporting under GHG Protocol, CDP, TCFD, or SBTi
- Comparing emission reduction strategies

## Commands

### `intro`

```bash
scripts/script.sh intro
```

Overview of GHG emissions — gases, GWP, global budget, and key sources.

### `scopes`

```bash
scripts/script.sh scopes
```

Scope 1, 2, 3 definitions with detailed category breakdown.

### `factors`

```bash
scripts/script.sh factors
```

Common emission factors: electricity, fuels, transport, materials.

### `calculation`

```bash
scripts/script.sh calculation
```

How to calculate emissions: activity data × emission factor methodology.

### `reporting`

```bash
scripts/script.sh reporting
```

Reporting frameworks: GHG Protocol, CDP, TCFD, SBTi, CSRD.

### `reduction`

```bash
scripts/script.sh reduction
```

Emission reduction strategies by sector: energy, transport, industry, buildings.

### `offsets`

```bash
scripts/script.sh offsets
```

Carbon offsets and credits — types, standards, additionality, controversies.

### `sectoral`

```bash
scripts/script.sh sectoral
```

Emissions by sector: energy, transport, industry, agriculture, buildings, waste.

### `help`

```bash
scripts/script.sh help
```

### `version`

```bash
scripts/script.sh version
```
