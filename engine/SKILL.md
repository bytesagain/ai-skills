# engine

Benchmark and profile script execution with timing. Use when measuring performance or comparing speeds.

## Commands

### `time`

```bash
scripts/script.sh time <cmd>
```

### `profile`

```bash
scripts/script.sh profile <script>
```

### `compare`

```bash
scripts/script.sh compare <cmd1 cmd2>
```

### `repeat`

```bash
scripts/script.sh repeat <count cmd>
```

### `stats`

```bash
scripts/script.sh stats <logfile>
```

### `memory`

```bash
scripts/script.sh memory <cmd>
```

## Data Storage

Data stored in `~/.local/share/engine/`.
