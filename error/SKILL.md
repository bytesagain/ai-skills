# error

Analyze error logs and generate error reports. Use when debugging failures or tracking error patterns.

## Commands

### `scan`

```bash
scripts/script.sh scan <file>
```

### `count`

```bash
scripts/script.sh count <file>
```

### `recent`

```bash
scripts/script.sh recent <file>
```

### `unique`

```bash
scripts/script.sh unique <file>
```

### `context`

```bash
scripts/script.sh context <file pattern>
```

### `report`

```bash
scripts/script.sh report <file>
```

## Data Storage

Data stored in `~/.local/share/error/`.
