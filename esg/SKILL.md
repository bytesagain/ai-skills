# ESG — ESG Reference

Quick-reference skill for Environmental, Social, and Governance criteria in investing and corporate reporting.

## When to Use

- Understanding ESG pillars and materiality assessment
- Selecting ESG reporting frameworks (GRI, SASB, ISSB, CSRD)
- Evaluating ESG ratings and data providers
- Implementing ESG integration in investment decisions
- Understanding regulatory requirements for ESG disclosure

## Commands

### `intro`

```bash
scripts/script.sh intro
```

Overview of ESG — the three pillars, history, and why it matters.

### `environmental`

```bash
scripts/script.sh environmental
```

Environmental pillar — climate, pollution, resources, biodiversity metrics.

### `social`

```bash
scripts/script.sh social
```

Social pillar — labor, human rights, community, diversity metrics.

### `governance`

```bash
scripts/script.sh governance
```

Governance pillar — board structure, ethics, transparency, executive pay.

### `frameworks`

```bash
scripts/script.sh frameworks
```

Reporting frameworks: GRI, SASB, ISSB/IFRS S1-S2, CSRD/ESRS, TCFD.

### `ratings`

```bash
scripts/script.sh ratings
```

ESG ratings — MSCI, S&P, Sustainalytics, ISS, CDP — methodology and divergence.

### `investing`

```bash
scripts/script.sh investing
```

ESG investment strategies — integration, screening, impact, stewardship.

### `regulation`

```bash
scripts/script.sh regulation
```

ESG regulation — EU Taxonomy, SFDR, SEC Climate Rule, global landscape.

### `help`

```bash
scripts/script.sh help
```

### `version`

```bash
scripts/script.sh version
```
