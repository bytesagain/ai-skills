# FakerJS
Generate realistic test data. Companies, products, dates, IPs, and more.
## Commands
- `company [n]` — Random company names
- `product [n]` — Random product names with prices
- `paragraph [n]` — Random paragraphs
- `date [range]` — Random dates
- `color [n]` — Random colors (hex)
- `ip [n]` — Random IP addresses
- `useragent [n]` — Random user agent strings
- `payload [type]` — JSON test payload (user/order/product)
## Usage Examples
```bash
fakerjs company 5
fakerjs product 3
fakerjs payload user
fakerjs color 10
```
