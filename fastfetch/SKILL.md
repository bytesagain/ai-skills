# Fastfetch

A maintained, feature-rich and performance oriented, neofetch like system information tool.

Inspired by [fastfetch-cli/fastfetch]([configured-endpoint]) (20,673+ GitHub stars).

## Commands

- `help` - Help
- `run` - Run
- `info` - Info
- `status` - Status

## Features

- Core functionality from fastfetch-cli/fastfetch

## Usage

Run any command: `fastfetch <command> [args]`

## Examples

```bash
fastfetch help
fastfetch run
```

## When to Use

- as part of a larger automation pipeline
- when you need quick fastfetch from the command line

## Output

Returns formatted output to stdout. Redirect to a file with `fastfetch run > output.txt`.

## Configuration

Set `FASTFETCH_DIR` environment variable to change the data directory. Default: `~/.local/share/fastfetch/`
