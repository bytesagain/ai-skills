# file

Manage, organize, and inspect files with batch operations. Use when finding duplicates, checking sizes, or batch renaming files.

## Commands

### `info`

```bash
scripts/script.sh info <path>
```

### `find`

```bash
scripts/script.sh find <pattern dir>
```

### `dupes`

```bash
scripts/script.sh dupes <dir>
```

### `large`

```bash
scripts/script.sh large <dir n>
```

### `recent`

```bash
scripts/script.sh recent <dir days>
```

### `tree`

```bash
scripts/script.sh tree <dir>
```

## Requirements

- bash 4.0+
