# FileWatch

File change watcher and notifier. Monitor files and directories for changes, trigger actions on modification, log change history, and track file versions.

## Quick Start

Run `filewatch help` for available commands and usage examples.

## Features

- Fast and lightweight — pure bash with embedded Python
- No external dependencies required
- Works on Linux and macOS

## Usage

```bash
filewatch help
```

Returns logs to stdout. Redirect to a file with `filewatch run > output.txt`.
