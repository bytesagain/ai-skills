# Filtration — Industrial & Water Filtration Technology

Quick-reference for filtration processes, membrane technologies, and filter system design.

## When to Use

- Selecting filtration methods for water or industrial processes
- Understanding membrane types (MF, UF, NF, RO)
- Sizing filter systems and calculating flux rates
- Troubleshooting fouling and performance degradation
- Designing multi-stage filtration trains

## Commands

### `intro`

```bash
scripts/script.sh intro
```

Overview of filtration — principles, driving forces, and classification.

### `membranes`

```bash
scripts/script.sh membranes
```

Membrane filtration types: microfiltration, ultrafiltration, nanofiltration, reverse osmosis.

### `media`

```bash
scripts/script.sh media
```

Filter media reference — sand, activated carbon, anthracite, cartridge, bag filters.

### `design`

```bash
scripts/script.sh design
```

Filtration system design — flux, TMP, recovery rates, and sizing calculations.

### `fouling`

```bash
scripts/script.sh fouling
```

Membrane fouling — causes, prevention, cleaning strategies, and CIP protocols.

### `industrial`

```bash
scripts/script.sh industrial
```

Industrial filtration applications — oil, chemicals, pharma, food & beverage.

### `water`

```bash
scripts/script.sh water
```

Water treatment filtration — drinking water, wastewater, and reuse systems.

### `compare`

```bash
scripts/script.sh compare
```

Filtration technology comparison — selection matrix and decision guide.

### `help`

```bash
scripts/script.sh help
```

### `version`

```bash
scripts/script.sh version
```
