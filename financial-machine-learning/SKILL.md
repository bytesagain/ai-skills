- `run` - Run
- `info` - Info

## Usage

Run any command: `financial-machine-learning <command> [args]`

## Configuration

Set `FINANCIAL_MACHINE_LEARNING_DIR` environment variable to change the data directory. Default: `~/.local/share/financial-machine-learning/`
