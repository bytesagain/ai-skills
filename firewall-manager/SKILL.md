# Simplewall

Simple tool to configure Windows Filtering Platform (WFP) which can configure network activity on your computer. ## Commands

- `help` - Help
- `run` - Run
- `info` - Info
- `status` - Status

## Features

- Core functionality from henrypp/firewall-manager

## Usage

Run any command: `firewall-manager <command> [args]`

```bash
firewall-manager help
firewall-manager run
```

## Output

Returns summaries to stdout. Redirect to a file with `firewall-manager run > output.txt`.
