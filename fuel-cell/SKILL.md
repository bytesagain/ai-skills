# Fuel Cell — Fuel Cell Technology Reference

Quick-reference for fuel cell types, electrochemistry, efficiency, and system design.

## When to Use

- Understanding fuel cell operating principles
- Comparing PEM, SOFC, MCFC, AFC, PAFC, and DMFC types
- Designing fuel cell systems and balance of plant
- Evaluating fuel cell applications (transport, stationary, portable)
- Calculating efficiency, power density, and degradation

## Commands

### `intro`

```bash
scripts/script.sh intro
```

Overview of fuel cells — electrochemistry, thermodynamics, and history.

### `types`

```bash
scripts/script.sh types
```

Fuel cell types comparison — PEM, SOFC, MCFC, AFC, PAFC, DMFC.

### `pem`

```bash
scripts/script.sh pem
```

PEM fuel cells — proton exchange membrane, transport applications.

### `sofc`

```bash
scripts/script.sh sofc
```

Solid oxide fuel cells — high temperature, stationary power.

### `efficiency`

```bash
scripts/script.sh efficiency
```

Fuel cell efficiency — Carnot limit bypass, real-world performance.

### `bop`

```bash
scripts/script.sh bop
```

Balance of plant — humidification, thermal management, power conditioning.

### `degradation`

```bash
scripts/script.sh degradation
```

Fuel cell degradation — mechanisms, mitigation, and lifetime prediction.

### `applications`

```bash
scripts/script.sh applications
```

Fuel cell applications — vehicles, stationary power, backup, portable.

### `help`

```bash
scripts/script.sh help
```

### `version`

```bash
scripts/script.sh version
```
