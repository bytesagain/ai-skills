# Geothermal — Geothermal Energy Reference

Quick-reference for geothermal energy systems, resource assessment, and power generation.

## When to Use

- Evaluating geothermal resource potential for a site
- Understanding hydrothermal vs EGS vs heat pump systems
- Designing geothermal power plants (flash, binary, dry steam)
- Planning ground-source heat pump installations
- Assessing environmental impacts of geothermal development

## Commands

### `intro`

```bash
scripts/script.sh intro
```

Overview of geothermal energy — Earth's heat, resource types, and global potential.

### `resources`

```bash
scripts/script.sh resources
```

Geothermal resource classification — temperature grades, reservoir types.

### `plants`

```bash
scripts/script.sh plants
```

Power plant types — dry steam, flash steam, binary cycle, and hybrid.

### `heatpump`

```bash
scripts/script.sh heatpump
```

Ground-source heat pumps — closed loop, open loop, COP calculations.

### `drilling`

```bash
scripts/script.sh drilling
```

Geothermal drilling — well design, directional drilling, and costs.

### `egs`

```bash
scripts/script.sh egs
```

Enhanced Geothermal Systems — hydraulic stimulation and hot dry rock.

### `environmental`

```bash
scripts/script.sh environmental
```

Environmental considerations — emissions, induced seismicity, water use.

### `economics`

```bash
scripts/script.sh economics
```

Geothermal economics — LCOE, capacity factors, and project development.

### `help`

```bash
scripts/script.sh help
```

### `version`

```bash
scripts/script.sh version
```
