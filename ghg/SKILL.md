# GHG — Greenhouse Gas Accounting Reference

Quick-reference for greenhouse gas emissions calculation, reporting, and reduction strategies.

## When to Use

- Calculating Scope 1, 2, and 3 GHG emissions
- Looking up emission factors for fuels, electricity, and activities
- Preparing GHG inventories per GHG Protocol standards
- Understanding global warming potentials (GWP) of different gases
- Reporting emissions for CDP, TCFD, or regulatory compliance

## Commands

### `intro`

```bash
scripts/script.sh intro
```

Overview of greenhouse gases — sources, GWP, and atmospheric concentrations.

### `scopes`

```bash
scripts/script.sh scopes
```

Scope 1, 2, 3 emissions — boundaries, definitions, and examples.

### `factors`

```bash
scripts/script.sh factors
```

Emission factors — fuel combustion, electricity, transport, and materials.

### `protocol`

```bash
scripts/script.sh protocol
```

GHG Protocol — standards, principles, and reporting frameworks.

### `calculate`

```bash
scripts/script.sh calculate
```

Calculation methods — activity data × emission factor, mass balance.

### `reporting`

```bash
scripts/script.sh reporting
```

Reporting frameworks — CDP, TCFD, SBTi, and regulatory requirements.

### `reduction`

```bash
scripts/script.sh reduction
```

Emission reduction strategies — decarbonization pathways by sector.

### `offsets`

```bash
scripts/script.sh offsets
```

Carbon offsets and credits — types, verification, and controversies.

### `help`

```bash
scripts/script.sh help
```

### `version`

```bash
scripts/script.sh version
```
