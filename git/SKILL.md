# git

Analyze git repositories with commit stats and branch tools. Use when reviewing history, comparing branches, or auditing contributors.

## Commands

### `log`

```bash
scripts/script.sh log <n>
```

### `stats`

```bash
scripts/script.sh stats
```

### `branches`

```bash
scripts/script.sh branches
```

### `diff-stat`

```bash
scripts/script.sh diff-stat
```

### `stale`

```bash
scripts/script.sh stale <days>
```

### `size`

```bash
scripts/script.sh size
```

## Requirements

- bash 4.0+
- git
