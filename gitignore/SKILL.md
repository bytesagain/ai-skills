# GitIgnore

Gitignore file generator. Create .gitignore files for any language or framework from curated templates. Supports Node, Python, Java, Go, Rust, and 50 more.

## Quick Start

Run `gitignore help` for available commands and usage examples.

## Features

- Fast and lightweight — pure bash with embedded Python
- No external dependencies required
- Works on Linux and macOS

## Usage

```bash
gitignore help
```

## Output

Returns formatted output to stdout. Redirect to a file with `gitignore run > output.txt`.

## Configuration

Set `GITIGNORE_DIR` environment variable to change the data directory. Default: `~/.local/share/gitignore/`
