# GitLog

Beautiful git log viewer and repository statistics. View formatted commit history, show contribution stats by author, analyze commit frequency patterns, find recent changes, and generate changelogs. Better git insights from your terminal.

## Quick Start

Run `gitlog help` for detailed usage information and available commands.
