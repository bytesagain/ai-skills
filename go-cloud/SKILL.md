# Go Cloud

The Go Cloud Development Kit (Go CDK): A library and tools for open cloud development in Go.

Inspired by [google/go-cloud]([configured-endpoint]) (9,873+ GitHub stars).

## Commands

- `help` - Help
- `run` - Run
- `info` - Info
- `status` - Status

## Features

- Core functionality from google/go-cloud

## Usage

Run any command: `go-cloud <command> [args]`

## Examples

```bash
go-cloud help
go-cloud run
```

## When to Use

- to automate go tasks in your workflow
- for batch processing cloud operations

## Output

Returns logs to stdout. Redirect to a file with `go-cloud run > output.txt`.

## Configuration

Set `GO_CLOUD_DIR` environment variable to change the data directory. Default: `~/.local/share/go-cloud/`
