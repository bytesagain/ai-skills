# Grapesjs

Free and Open source Web Builder Framework. Next generation tool for building templates without coding

Inspired by [GrapesJS/grapesjs]([configured-endpoint]) (25,612+ GitHub stars).

## Commands

- `help` - Help
- `run` - Run
- `info` - Info
- `status` - Status

## Features

- Core functionality from GrapesJS/grapesjs

## Usage

Run any command: `grapesjs <command> [args]`

## Examples

```bash
grapesjs help
grapesjs run
```

## When to Use

- to automate grapesjs tasks in your workflow
- for batch processing grapesjs operations

## Output

Returns reports to stdout. Redirect to a file with `grapesjs run > output.txt`.

## Configuration

Set `GRAPESJS_DIR` environment variable to change the data directory. Default: `~/.local/share/grapesjs/`
