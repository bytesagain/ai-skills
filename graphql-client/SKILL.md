# Apollo Client

The industry-leading GraphQL client for TypeScript, JavaScript, React, Vue, Angular, and more. Apollo Client delivers powerful caching, intuitive APIs, and complete developer tools to accelerate your app development. ## Commands

- `help` - Help
- `run` - Run
- `info` - Info
- `status` - Status

## Features

- Core functionality from apollographql/graphql-client

## Usage

Run any command: `graphql-client <command> [args]`

## Output

Returns reports to stdout. Redirect to a file with `graphql-client run > output.txt`.
