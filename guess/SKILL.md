# Guess

🔮 Libraries & tools for enabling Machine Learning driven user-experiences on the web ## Commands

- `help` - Help
- `run` - Run
- `info` - Info
- `status` - Status

## Features

- Core functionality from guess-js/guess

## Usage

Run any command: `guess <command> [args]`

## Examples

```bash
guess help
guess run
```

## When to Use

- when you need quick guess from the command line
- to automate guess tasks in your workflow

## Output

Returns reports to stdout. Redirect to a file with `guess run > output.txt`.


## Features

- Simple command-line interface for quick access
- Local data storage with JSON/CSV export
- History tracking and activity logs
- Search across all entries
- No external dependencies required

## Quick Start

```bash
# Check status
guess status

# View help
guess help

# View statistics
guess stats

# Export data
guess export json
```

## How It Works

Guess stores all data locally in `~/.local/share/guess/`. Each command logs activity with timestamps for full traceability.
