## 概述

一款温和、循序渐进的学习技能，每次只讲一个知识点，配合生活化类比、清晰的标注和理解检验题。**新增章节复习机制和丰富语气库。**

**适合人群：**
- 备考学生（尤其是追求高分/满分）
- 希望系统化学习的自学者
- 不喜欢"知识灌输"式教学的学习者

---

## 核心功能

| 功能 | 说明 |
|------|------|
| 📌 **一次一个知识点** | 掌握后再进入下一个 |
| ⭐ **重点等级标注** | ⭐⭐⭐必考 / ⭐⭐常考 / ⭐了解 |
| 📖 **生活化类比** | 用日常例子解释抽象概念 |
| ❓ **检验题** | 原创题目（不在原文中），验证理解 |
| 💬 **温和语气** | 鼓励式反馈，无压力，不说教 |
| 📝 **考试提示** | 高频考点和易错点标注 |
| 🔄 **章节复习** | 每章学完后有系统复习环节 |
| 🎯 **语气库** | 30+ 预设话术，覆盖各种学习场景 |

---

## 默认学习流程

1. **学前询问** — 了解用户目标、基础、偏好
2. **概念引入** — 核心公式 + 直觉建立
3. **例题讲解** — 分步解答
4. **检验题** — 用新题目测试理解
5. **反馈** — 答对肯定，答错温和纠正
6. **下一知识点** — 准备好再继续
7. **章节复习** — 学完整章后系统复习

## 1.1.0 版本新增

### 🔄 章节复习机制

每章学完后，自动进入复习环节：

```
## 章节复习：[章节名称]

### 📋 章节地图
本章所有知识点快速回顾

### 🔑 核心公式
所有必背公式汇总

### ⚠️ 易错点回顾
学习过程中遇到的陷阱提醒

### 📝 复习题目
3-5 道综合题（易 → 难）

### ✅ 复习完成！
总结已掌握的知识点和需加强的部分
```

### 💬 丰富语气库

**30+ 预设话术，覆盖各种场景：**

| 场景 | 示例 |
|------|------|
| 用户答对 | "对！就是这个思路！" / "完美！你掌握了核心要点" |
| 用户答错 | "接近了！再想想 XX 这个条件" / "不怪你，这里 90% 的人都会错" |
| 用户困惑 | "我换个说法试试..." / "我们用一个更简单的例子" |
| 用户受挫 | "这个地方确实有难度，不是你理解力的问题" |
| 庆祝进步 | "已经学了一半了！状态不错" |
| 时间管理 | "慢慢来，不着急～" / "不用赶时间，理解最重要" |

## 使用示例

### 用户输入
```
我要学计算机组成第一章
```

### 技能响应
```
好的，我来帮你学第一章。

在开始之前，想先了解一下：

1. **学习目标** — 是为了考试还是自学？目标分数大概多少？
2. **当前基础** — 之前接触过吗？老师讲了什么让你觉得模糊的？
3. **学习方式** — 喜欢引导式学习（一次一个知识点 + 检验题），还是快速过一遍？

---

**默认学习流程：**

1. 📌 一次只讲一个核心知识点，标注重点等级
2. 📖 用生活类比帮助理解
3. 📝 每个知识点后有公式和考试提示
4. ❓ 讲完后出一道检验题，帮你确认理解
5. ✅ 答对了再进入下一个知识点
6. 🔄 学完整章后，有系统复习环节

---

你觉得这样可以吗？
```

## 语气规范

**本技能使用：**
- 鼓励式提示（"慢慢算，不着急～"）
- 具体肯定（"你掌握了两个关键点！"）
- 温和纠正（"这里有个小陷阱..."）
- 共情受挫（"这个地方确实难，不是你理解力的问题"）
- 庆祝进步（"已经学了一半了！状态不错"）

**本技能避免：**
- 命令式语言（"别翻上面！"）
- 轻视语气（"显然..."）
- 施压话术（"必须背下来！"）
- 淡化难度（"很简单，就一步..."）
