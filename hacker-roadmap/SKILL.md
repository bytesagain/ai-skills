# Hacker Roadmap

A collection of hacking tools, resources and references to practice ethical hacking.

Inspired by [sundowndev/hacker-roadmap]([configured-endpoint]) (15,130+ GitHub stars).

## Commands

- `help` - Help
- `run` - Run
- `info` - Info
- `status` - Status

## Features

- Core functionality from sundowndev/hacker-roadmap

## Usage

Run any command: `hacker-roadmap <command> [args]`

## Output

Returns results to stdout. Redirect to a file with `hacker-roadmap run > output.txt`.

## Configuration

Set `HACKER_ROADMAP_DIR` environment variable to change the data directory. Default: `~/.local/share/hacker-roadmap/`
