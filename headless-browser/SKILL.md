# Phantomjs

Scriptable Headless Browser ## Commands

- `help` - Help
- `run` - Run
- `info` - Info
- `status` - Status

## Features

- Core functionality from ariya/headless-browser

## Usage

Run any command: `headless-browser <command> [args]`

## Examples

```bash
headless-browser help
headless-browser run
```

## When to Use

- when you need quick headless browser from the command line
- to automate headless tasks in your workflow

## Output

Returns reports to stdout. Redirect to a file with `headless-browser run > output.txt`.


## Features

- Simple command-line interface for quick access
- Local data storage with JSON/CSV export
- History tracking and activity logs
- Search across all entries
- No external dependencies required

## Quick Start

```bash
# Check status
headless-browser status

# View help
headless-browser help

# View statistics
headless-browser stats

# Export data
headless-browser export json
```

## How It Works

Headless Browser stores all data locally in `~/.local/share/headless-browser/`. Each command logs activity with timestamps for full traceability.
