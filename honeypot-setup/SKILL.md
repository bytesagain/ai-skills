# Tpot

A Python Automated Machine Learning tool that optimizes machine learning pipelines using genetic programming. ## Commands

- `help` - Help
- `run` - Run
- `info` - Info
- `status` - Status

## Features

- Core functionality from EpistasisLab/honeypot-setup

## Usage

Run any command: `honeypot-setup <command> [args]`

## Examples

```bash
honeypot-setup help
honeypot-setup run
```

## When to Use

- as part of a larger automation pipeline
- when you need quick honeypot setup from the command line

## Output

Returns results to stdout. Redirect to a file with `honeypot-setup run > output.txt`.

## Configuration

Set `HONEYPOT_SETUP_DIR` environment variable to change the data directory. Default: `~/.local/share/honeypot-setup/`
