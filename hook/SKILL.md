# Hook — Hooks & Lifecycle Callbacks Reference

Quick-reference for hook patterns across Git, React, webhooks, and system-level event hooks.

## When to Use

- Setting up Git hooks (pre-commit, pre-push, commit-msg)
- Understanding React hook rules and patterns
- Designing webhook integrations for event-driven systems
- Implementing lifecycle hooks in frameworks
- Choosing hook management tools (Husky, lefthook)

## Commands

### `intro`

```bash
scripts/script.sh intro
```

Overview of hook patterns — what hooks are, where they appear, and why they matter.

### `git`

```bash
scripts/script.sh git
```

Git hooks — pre-commit, commit-msg, pre-push, and server-side hooks.

### `react`

```bash
scripts/script.sh react
```

React hooks — useState, useEffect, useMemo, custom hooks, and rules.

### `webhook`


Webhooks — design, delivery, retry logic, and security verification.

### `tools`


Hook management tools — Husky, lefthook, lint-staged, and pre-commit.

### `patterns`
