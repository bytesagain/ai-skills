# http

Send HTTP requests and analyze responses with timing. Use when testing web endpoints, checking headers, or measuring latency.

## Commands

### `get`

```bash
scripts/script.sh get <url>
```

### `post`

```bash
scripts/script.sh post <url body>
```

### `head`

```bash
scripts/script.sh head <url>
```

### `time`

```bash
scripts/script.sh time <url>
```

### `status`

```bash
scripts/script.sh status <url>
```

### `follow`

```bash
scripts/script.sh follow <url>
```

## Requirements

- bash 4.0+
- curl
