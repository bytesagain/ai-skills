# Hydrogen — Hydrogen Energy Reference

Quick-reference for hydrogen production, storage, transport, and applications in the energy transition.

## When to Use

- Evaluating hydrogen production methods and their carbon intensity
- Understanding electrolysis technologies (PEM, alkaline, SOEC)
- Planning hydrogen storage and transport infrastructure
- Comparing hydrogen vs electrification for different sectors
- Assessing hydrogen project economics and LCOH

## Commands

### `intro`

```bash
scripts/script.sh intro
```

Overview of hydrogen energy — properties, color spectrum, and role in decarbonization.

### `production`

```bash
scripts/script.sh production
```

Hydrogen production methods — SMR, electrolysis, pyrolysis, and biomass.

### `electrolysis`

```bash
scripts/script.sh electrolysis
```

Electrolysis deep dive — PEM, alkaline, SOEC technology comparison.

### `storage`

```bash
scripts/script.sh storage
```

Hydrogen storage — compressed gas, liquid, metal hydrides, and underground.

### `transport`

```bash
scripts/script.sh transport
```

Hydrogen transport — pipelines, tube trailers, liquid tankers, and ammonia carriers.

### `applications`

```bash
scripts/script.sh applications
```

End-use applications — industry, transport, power, and heating.

### `safety`

```bash
scripts/script.sh safety
```

Hydrogen safety — properties, risks, codes, and standards.

### `economics`

```bash
scripts/script.sh economics
```

Hydrogen economics — LCOH, cost drivers, and market projections.

### `help`

```bash
scripts/script.sh help
```

### `version`

```bash
scripts/script.sh version
```
