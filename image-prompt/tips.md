# 💡 Image Prompt Tips

1. **Subject first, modifiers after**: Prompt structure follows "Subject → Environment → Style → Lighting → Quality" — subject at the front gets highest weight.

2. **Keep MJ prompts short**: Midjourney responds best to 30-75 words. Longer prompts often produce worse results.

3. **Use SD weights**: In Stable Diffusion, `(keyword:1.3)` boosts weight, `[keyword]` reduces it — precise control over elements.

4. **Always write negative prompts for SD**: Common must-haves: `blurry, low quality, deformed, extra limbs, watermark`.

5. **Stack enhance calls**: Generate a basic prompt, then run `enhance` 2-3 times for progressive refinement.

6. **Batch for style exploration**: Same subject, 5-10 style variants — pick the best one without manual rewriting.

7. **Check the style library first**: Not sure about style names? Run `style` to browse categories before guessing keywords.

8. **English prompts work best**: All AI image tools perform better with English prompts — the tool outputs English by default.
