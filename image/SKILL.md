# image

Inspect and process image files with metadata and conversion. Use when checking dimensions, extracting EXIF data, or batch processing images.

## Commands

### `info`

```bash
scripts/script.sh info <file>
```

### `size`

```bash
scripts/script.sh size <file>
```

### `exif`

```bash
scripts/script.sh exif <file>
```

### `list`

```bash
scripts/script.sh list <dir>
```

### `convert`

```bash
scripts/script.sh convert <in out>
```

### `thumb`

```bash
scripts/script.sh thumb <file size>
```

## Requirements

- bash 4.0+
