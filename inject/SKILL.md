# Inject — Dependency Injection Reference

Quick-reference for dependency injection patterns, IoC containers, and inversion of control principles.

## When to Use

- Implementing dependency injection in your application
- Choosing between constructor, property, and method injection
- Setting up IoC containers (Spring, .NET DI, InversifyJS, etc.)
- Refactoring tightly-coupled code for testability
- Understanding service lifetimes (singleton, scoped, transient)

## Commands

### `intro`

```bash
scripts/script.sh intro
```

Overview of dependency injection — what, why, and the SOLID connection.

### `patterns`

```bash
scripts/script.sh patterns
```

DI patterns — constructor, property, method, and interface injection.

### `containers`

```bash
scripts/script.sh containers
```

IoC containers — Spring, .NET DI, InversifyJS, tsyringe, and more.

### `lifetimes`

```bash
scripts/script.sh lifetimes
```

Service lifetimes — singleton, scoped, transient, and when to use each.

### `testing`

```bash
scripts/script.sh testing
```

DI for testability — mocking, faking, and test doubles with injection.

### `antipatterns`

```bash
scripts/script.sh antipatterns
```

DI anti-patterns — service locator, captive dependencies, and over-injection.

### `typescript`

```bash
scripts/script.sh typescript
```

DI in TypeScript/JavaScript — decorators, tokens, and manual wiring.

### `frameworks`

```bash
scripts/script.sh frameworks
```

Framework-specific DI — Angular, NestJS, Spring Boot, ASP.NET Core.

### `help`

```

### `version`

```
