# It Tools

Collection of handy online tools for developers, with great UX. 

Inspired by [CorentinTh/it-tools]([configured-endpoint]) (37,561+ GitHub stars).

## Commands

- `help` - Help
- `run` - Run
- `info` - Info
- `status` - Status

## Features

- Core functionality from CorentinTh/it-tools

## Usage

Run any command: `it-tools <command> [args]`

## Examples

```bash
it-tools help
it-tools run
```

## When to Use

- as part of a larger automation pipeline
- when you need quick it tools from the command line

## Output

Returns reports to stdout. Redirect to a file with `it-tools run > output.txt`.

## Configuration

Set `IT_TOOLS_DIR` environment variable to change the data directory. Default: `~/.local/share/it-tools/`
