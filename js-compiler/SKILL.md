# Oxc

⚓ A collection of high-performance JavaScript tools. ## Commands

- `help` - Help
- `run` - Run
- `info` - Info
- `status` - Status

## Features

- Core functionality from js-compiler-project/js-compiler

## Usage

Run any command: `js-compiler <command> [args]`

## Examples

```bash
js-compiler help
js-compiler run
```

## When to Use

- to automate js tasks in your workflow
- for batch processing compiler operations

## Output

Returns results to stdout. Redirect to a file with `js-compiler run > output.txt`.

## Configuration

Set `JS_COMPILER_DIR` environment variable to change the data directory. Default: `~/.local/share/js-compiler/`
