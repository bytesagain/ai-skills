# json-formatter

JSON格式化、验证、转换(JSON↔YAML/XML/CSV)、压缩、对比、Schema生成。JSON formatter, validator, converter, minifier, diff, schema generator.

## 推荐工作流

```
需求分析 → 选择命令 → 输入描述 → 获取结果 → 调整优化
```

## 可用命令

- **format** — format
- **validate** — validate
- **convert** — convert
- **minify** — minify
- **diff** — diff
- **schema** — schema


## 专业建议

- 对象** `{}` — 键值对集合，键必须是双引号字符串
- 数组** `[]` — 有序值列表
- 字符串** `"hello"` — 必须双引号
- 数字** `42`, `3.14`, `-1`, `1e10`
- 布尔** `true`, `false`
