# Jsoncrackcom

✨ practical and open-source visualization application that transforms various data formats, such as JSON, YAML, XML and CSV into interactive graphs.

Inspired by [AykutSarac/jsoncrack.com]([configured-endpoint]) (43,427+ GitHub stars).

## Commands

- `help` - Help
- `run` - Run
- `info` - Info
- `status` - Status

## Features

- Core functionality from AykutSarac/jsoncrack.com

## Usage

Run any command: `jsoncrackcom <command> [args]`

jsoncrackcom help
jsoncrackcom run

- for batch processing jsoncrackcom operations

## Output

Returns summaries to stdout. Redirect to a file with `jsoncrackcom run > output.txt`.

## Configuration

Set `JSONCRACKCOM_DIR` environment variable to change the data directory. Default: `~/.local/share/jsoncrackcom/`
