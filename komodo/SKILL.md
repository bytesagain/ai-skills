# Komodo

🦎 a tool to build and deploy software on many servers 🦎

Inspired by [moghtech/komodo]([configured-endpoint]) (10,534+ GitHub stars).

## Commands

- `help` - Help
- `run` - Run
- `info` - Info
- `status` - Status

## Features

- Core functionality from moghtech/komodo

## Usage

Run any command: `komodo <command> [args]`

## Examples

```bash
komodo help
komodo run
```

## When to Use

- as part of a larger automation pipeline
- when you need quick komodo from the command line

## Output

Returns structured data to stdout. Redirect to a file with `komodo run > output.txt`.

## Configuration

Set `KOMODO_DIR` environment variable to change the data directory. Default: `~/.local/share/komodo/`
