# Langflow

Langflow is a powerful tool for building and deploying AI-powered agents and workflows.

Inspired by [langflow-ai/langflow]([configured-endpoint]) (145,651+ GitHub stars).

## Commands

- `help` - Help
- `run` - Run
- `info` - Info
- `status` - Status

## Features

- Core functionality from langflow-ai/langflow

## Usage

Run any command: `langflow <command> [args]`

## Examples

```bash
langflow help
langflow run
```

## When to Use

- to automate langflow tasks in your workflow
- for batch processing langflow operations

## Output

Returns structured data to stdout. Redirect to a file with `langflow run > output.txt`.

## Configuration

Set `LANGFLOW_DIR` environment variable to change the data directory. Default: `~/.local/share/langflow/`
