# Lifecycle — Product Lifecycle Analysis Reference

Quick-reference skill for lifecycle assessment (LCA) methodology, environmental impact analysis, and circular economy principles.

## When to Use

- Conducting a cradle-to-grave product assessment
- Understanding LCA phases and methodology (ISO 14040/14044)
- Evaluating environmental impact categories
- Designing products for circular economy
- Comparing material and process environmental footprints

## Commands

### `intro`

```bash
scripts/script.sh intro
```

Overview of lifecycle assessment — history, ISO standards, core framework.

### `phases`

```bash
scripts/script.sh phases
```

The four LCA phases: Goal & Scope, Inventory Analysis, Impact Assessment, Interpretation.

### `impacts`

```bash
scripts/script.sh impacts
```

Environmental impact categories: GWP, acidification, eutrophication, ozone depletion, etc.

### `inventory`

```bash
scripts/script.sh inventory
```

Life Cycle Inventory (LCI) — data collection, system boundaries, allocation methods.

### `circular`

```bash
scripts/script.sh circular
```

Circular economy strategies: R-strategies, design for disassembly, material loops.

### `carbon`

```bash
scripts/script.sh carbon
```

Carbon footprint methodology — Scope 1/2/3 emissions, GHG Protocol, calculation methods.

### `materials`

```bash
scripts/script.sh materials
```

Embodied energy and environmental profiles of common engineering materials.

### `cases`

```bash
scripts/script.sh cases
```

Real-world LCA case studies comparing product alternatives.

### `help`

```bash
scripts/script.sh help
```

### `version`

```bash
scripts/script.sh version
```
