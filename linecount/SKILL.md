# LineCount

Source code line counter and statistics. Count lines of code by language, show file breakdown, exclude comments and blanks, and compare codebases. Quick project size analysis.

## Quick Start

Run `linecount help` for available commands and usage examples.

## Features

- Fast and lightweight — pure bash with embedded Python
- No external dependencies required
- Works on Linux and macOS

## Usage

```bash
linecount help
```
