# Langflow

Langflow is a powerful tool for building and deploying AI-powered agents and workflows. ## Commands

- `help` - Help
- `run` - Run
- `info` - Info
- `status` - Status

## Features

- Core functionality from llm-flow-ai/llm-flow

## Usage

Run any command: `llm-flow <command> [args]`

## Examples

```bash
llm-flow help
llm-flow run
```

## When to Use

- as part of a larger automation pipeline
- when you need quick llm flow from the command line

## Output

Returns reports to stdout. Redirect to a file with `llm-flow run > output.txt`.

## Configuration

Set `LLM_FLOW_DIR` environment variable to change the data directory. Default: `~/.local/share/llm-flow/`
