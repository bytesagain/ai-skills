# log

Analyze, search, and manage log files with pattern matching and stats. Use when tailing logs, finding errors, or computing log statistics.

## Commands

### `tail`

```bash
scripts/script.sh tail <file lines>
```

### `search`

```bash
scripts/script.sh search <pattern file>
```

### `errors`

```bash
scripts/script.sh errors <file>
```

### `stats`

```bash
scripts/script.sh stats <file>
```

### `filter`

```bash
scripts/script.sh filter <file level>
```

### `between`

```bash
scripts/script.sh between <file start end>
```

## Data Storage

Data stored in `~/.local/share/log/`.
