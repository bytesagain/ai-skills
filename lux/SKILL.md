# Lux

👾 Fast and simple video download library and CLI tool written in Go

Inspired by [iawia002/lux]([configured-endpoint]) (30,948+ GitHub stars).

## Commands

- `help` - Help
- `run` - Run
- `info` - Info
- `status` - Status

## Features

- Core functionality from iawia002/lux

## Usage

Run any command: `lux <command> [args]`

## Output

Returns logs to stdout. Redirect to a file with `lux run > output.txt`.

## Configuration

Set `LUX_DIR` environment variable to change the data directory. Default: `~/.local/share/lux/`
