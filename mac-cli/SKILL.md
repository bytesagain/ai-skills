# Mac Cli

 macOS command line tool for developers – The ultimate tool to manage your Mac. It provides a huge set of command line commands that automatize the usage of your Mac.

Inspired by [guarinogabriel/Mac-CLI]([configured-endpoint]) (9,049+ GitHub stars).

## Commands

- `help` - Help
- `run` - Run
- `info` - Info
- `status` - Status

## Features

- Core functionality from guarinogabriel/Mac-CLI

## Usage

Run any command: `mac-cli <command> [args]`

## Examples

```bash
mac-cli help
mac-cli run
```

## When to Use

- to automate mac tasks in your workflow
- for batch processing cli operations

## Output

Returns results to stdout. Redirect to a file with `mac-cli run > output.txt`.

## Configuration

Set `MAC_CLI_DIR` environment variable to change the data directory. Default: `~/.local/share/mac-cli/`
