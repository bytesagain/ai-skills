# MACD — Moving Average Convergence Divergence

Calculate and interpret MACD for any crypto or stock asset. Analyze signal line crossovers, histogram momentum, divergence patterns, and proven MACD trading strategies.

## Commands

### calculate
Calculate MACD from price data with standard (12,26,9) or custom parameters.
```bash
bash scripts/script.sh calculate 12 26 9
```

### interpret
Interpret MACD, signal, and histogram values with actionable signals.
```bash
bash scripts/script.sh interpret 1.25 0.80
```

### crossover
Explain MACD/signal line crossover patterns and trading rules.
```bash
bash scripts/script.sh crossover
```

### histogram
Deep dive into MACD histogram analysis and momentum reading.
```bash
bash scripts/script.sh histogram
```

### strategies
Proven MACD trading strategies with entry/exit rules.
```bash
bash scripts/script.sh strategies
```

### help
Show all commands.
```bash
bash scripts/script.sh help
```
