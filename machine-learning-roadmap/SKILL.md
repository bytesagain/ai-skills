# Machine Learning Roadmap

A roadmap connecting many of the most important concepts in machine learning, how to learn them and what tools to use to perform them.

Inspired by [mrdbourke/machine-learning-roadmap]([configured-endpoint]) (7,797+ GitHub stars).

## Commands

- `help` - Help
- `run` - Run
- `info` - Info
- `status` - Status

## Features

- Core functionality from mrdbourke/machine-learning-roadmap

## Usage

Run any command: `machine-learning-roadmap <command> [args]`

## Examples

```bash
machine-learning-roadmap help
machine-learning-roadmap run
```

## When to Use

- as part of a larger automation pipeline
- when you need quick machine learning roadmap from the command line

## Output

Returns structured data to stdout. Redirect to a file with `machine-learning-roadmap run > output.txt`.

## Configuration

Set `MACHINE_LEARNING_ROADMAP_DIR` environment variable to change the data directory. Default: `~/.local/share/machine-learning-roadmap/`
