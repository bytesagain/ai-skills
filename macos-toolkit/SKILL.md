# Mac Cli

 macOS command line tool for developers – The ultimate tool to manage your Mac. It provides a huge set of command line commands that automatize the usage of your Mac. ## Commands

- `help` - Help
- `run` - Run
- `info` - Info
- `status` - Status

## Features

- Core functionality from guarinogabriel/Mac-CLI

## Usage

Run any command: `macos-toolkit <command> [args]`

## Examples

```bash
macos-toolkit help
macos-toolkit run
```

## When to Use

- when you need quick macos toolkit from the command line
- to automate macos tasks in your workflow

## Output

Returns results to stdout. Redirect to a file with `macos-toolkit run > output.txt`.
