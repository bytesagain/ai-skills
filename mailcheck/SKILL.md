# MailCheck

Email deliverability checker. Validate email addresses, check MX records, verify SPF and DKIM records, test SMTP connectivity, and diagnose email delivery issues.

## Quick Start

Run `mailcheck help` for available commands and usage examples.

## Features

- Fast and lightweight — pure bash with embedded Python
- No external dependencies required
- Works on Linux and macOS

## Usage

```bash
mailcheck help
```
