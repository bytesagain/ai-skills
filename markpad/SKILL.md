# MarkPad

Markdown editor toolkit. Preview markdown as formatted text, convert markdown to HTML, validate markdown syntax, generate table of contents, count words and reading time, and create markdown templates. Write better docs faster.

## Quick Start

Run `markpad help` for detailed usage information and available commands.
