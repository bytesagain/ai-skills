# MathBox
Math tools for your terminal. Stats, primes, sequences, and formulas.
## Commands
- `stats <values>` — Statistics (mean, median, mode, stdev)
- `prime <n>` — Check if number is prime
- `primes <n>` — List first N primes
- `factor <n>` — Prime factorization
- `fib <n>` — Fibonacci sequence
- `gcd <a> <b>` — Greatest common divisor
- `eval <expression>` — Evaluate math expression
## Usage Examples
```bash
mathbox stats "10,20,30,40,50"
mathbox prime 97
mathbox factor 360
mathbox fib 15
```
