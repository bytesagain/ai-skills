# Meal Planner

膳食规划助手。周计划、热量计算、食谱推荐、购物清单、饮食方案(减脂/增肌/糖尿病)、预算控制。Meal planner with weekly plans, calorie calculator, recipes, shopping lists, diet plans, budget meals. 膳食、营养、食谱、减肥。

## 推荐工作流

```
需求分析 → 选择命令 → 输入描述 → 获取结果 → 调整优化
```

## 命令列表

| 命令 | 功能 |
|------|------|
| `plan` | plan |
| `calories` | calories |
| `recipe` | recipe |
| `shopping` | shopping |
| `diet` | diet |
| `budget` | budget |


## 专业建议

- 成年人: 1800-2200kcal/天
- 蛋白质: 体重(kg) × 1.0-1.6g
- 碳水: 总热量45-65%
- 脂肪: 总热量20-35%
- 每餐蛋白质+碳水+蔬菜
