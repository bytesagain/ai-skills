# Nebula

A flexible overlay networking tool with a focus on performance, simplicity and security ## Commands

- `help` - Help
- `run` - Run
- `info` - Info
- `status` - Status

## Features

- Core functionality from slackhq/mesh-network

## Usage

Run any command: `mesh-network <command> [args]`

## Examples

```bash
mesh-network help
mesh-network run
```

## When to Use

- for batch processing network operations
- as part of a larger automation pipeline

## Output

Returns structured data to stdout. Redirect to a file with `mesh-network run > output.txt`.


## Features

- Simple command-line interface for quick access
- Local data storage with JSON/CSV export
- History tracking and activity logs
- Search across all entries
- No external dependencies required

## Quick Start

```bash
# Check status
mesh-network status

# View help
mesh-network help

# View statistics
mesh-network stats

# Export data
mesh-network export json
```

## How It Works

Mesh Network stores all data locally in `~/.local/share/mesh-network/`. Each command logs activity with timestamps for full traceability.
