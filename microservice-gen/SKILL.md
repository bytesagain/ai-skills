# Go Zero

A cloud-native Go microservices framework with cli tool for productivity. ## Commands

- `help` - Help
- `run` - Run
- `info` - Info
- `status` - Status

## Features

- Core functionality from zeromicro/microservice-gen

## Usage

Run any command: `microservice-gen <command> [args]`

## Examples

```bash
microservice-gen help
microservice-gen run
```

## When to Use

- for batch processing gen operations
- as part of a larger automation pipeline

## Output

Returns formatted output to stdout. Redirect to a file with `microservice-gen run > output.txt`.


## Features

- Simple command-line interface for quick access
- Local data storage with JSON/CSV export
- History tracking and activity logs
- Search across all entries
- No external dependencies required

## Quick Start

```bash
# Check status
microservice-gen status

# View help
microservice-gen help

# View statistics
microservice-gen stats

# Export data
microservice-gen export json
```

## How It Works

Microservice Gen stores all data locally in `~/.local/share/microservice-gen/`. Each command logs activity with timestamps for full traceability.
