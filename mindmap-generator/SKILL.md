# mindmap-generator

思维导图生成工具。Markdown缩进输出、头脑风暴、知识组织、大纲转化、展开细化、模板库。Mindmap generator with Markdown indented output, brainstorming, knowledge organization, outline conversion, expansion, templates.

## 推荐工作流

```
需求分析 → 选择命令 → 输入描述 → 获取结果 → 调整优化
```

## 可用命令

- **create** — create
- **brainstorm** — brainstorm
- **organize** — organize
- **outline** — outline
- **expand** — expand
- **template** — template
- **swot** — swot
- **okr** — okr


## 专业建议

- 中心明确** — 核心主题放中间
- 分支不超过7个** — 认知负荷理论
- 关键词优先** — 用词语不用句子
- 层级清晰** — 一般3-4层就够
- 颜色区分** — 不同分支用不同颜色
