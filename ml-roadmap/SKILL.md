# Machine Learning Roadmap

A roadmap connecting many of the most important concepts in machine learning, how to learn them and what tools to use to perform them. ## Commands

- `help` - Help
- `run` - Run
- `info` - Info
- `status` - Status

## Features

- Core functionality from mrdbourke/ml-roadmap

## Usage

Run any command: `ml-roadmap <command> [args]`

## Examples

```bash
ml-roadmap help
ml-roadmap run
```

## When to Use

- as part of a larger automation pipeline
- when you need quick ml roadmap from the command line

## Output

Returns summaries to stdout. Redirect to a file with `ml-roadmap run > output.txt`.

## Configuration

Set `ML_ROADMAP_DIR` environment variable to change the data directory. Default: `~/.local/share/ml-roadmap/`
