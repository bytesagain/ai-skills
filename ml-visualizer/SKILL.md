# Yellowbrick

Visual analysis and diagnostic tools to help machine learning model selection. ## Commands

- `help` - Help
- `run` - Run
- `info` - Info
- `status` - Status

## Features

- Core functionality from DistrictDataLabs/ml-visualizer

## Usage

Run any command: `ml-visualizer <command> [args]`

## Examples

```bash
ml-visualizer help
ml-visualizer run
```

## When to Use

- to automate ml tasks in your workflow
- for batch processing visualizer operations

## Output

Returns structured data to stdout. Redirect to a file with `ml-visualizer run > output.txt`.

## Configuration

Set `ML_VISUALIZER_DIR` environment variable to change the data directory. Default: `~/.local/share/ml-visualizer/`
