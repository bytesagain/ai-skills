# Mlfinlab

MlFinLab helps portfolio managers and traders who want to use the power of machine learning by providing reproducible, interpretable, and easy to use tools. 

Inspired by [hudson-and-thames/mlfinlab]([configured-endpoint]) (4,591+ GitHub stars).

## Commands

- `help` - Help
- `run` - Run
- `info` - Info
- `status` - Status

## Features

- Core functionality from hudson-and-thames/mlfinlab

## Usage

Run any command: `mlfinlab <command> [args]`

## Examples

```bash
mlfinlab help
mlfinlab run
```

## When to Use

- to automate mlfinlab tasks in your workflow
- for batch processing mlfinlab operations

## Output

Returns results to stdout. Redirect to a file with `mlfinlab run > output.txt`.

## Configuration

Set `MLFINLAB_DIR` environment variable to change the data directory. Default: `~/.local/share/mlfinlab/`
