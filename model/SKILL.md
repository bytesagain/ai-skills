# model

Evaluate ML model predictions with accuracy, precision, and confusion matrix. Use when comparing predictions, computing metrics, or benchmarking.

## Commands

### `accuracy`

```bash
scripts/script.sh accuracy <pred actual>
```

### `compare`

```bash
scripts/script.sh compare <f1 f2>
```

### `confusion`

```bash
scripts/script.sh confusion <pred actual>
```

### `stats`

```bash
scripts/script.sh stats <file>
```

### `split`

```bash
scripts/script.sh split <file ratio>
```

### `describe`

```bash
scripts/script.sh describe <file>
```

## Data Storage

Data stored in `~/.local/share/model/`.
