# Nebula

A flexible overlay networking tool with a focus on performance, simplicity and security

Inspired by [slackhq/nebula]([configured-endpoint]) (17,108+ GitHub stars).

## Commands

- `help` - Help
- `run` - Run
- `info` - Info
- `status` - Status

## Features

- Core functionality from slackhq/nebula

## Usage

Run any command: `nebula <command> [args]`

## Output

Returns formatted output to stdout. Redirect to a file with `nebula run > output.txt`.

## Configuration

Set `NEBULA_DIR` environment variable to change the data directory. Default: `~/.local/share/nebula/`
