# net

Diagnose network issues with port scanning, DNS lookup, and speed tests. Use when checking connectivity, scanning ports, or tracing routes.

## Commands

### `scan`

```bash
scripts/script.sh scan <host>
```

### `ports`

```bash
scripts/script.sh ports
```

### `dns`

```bash
scripts/script.sh dns <domain>
```

### `interfaces`

```bash
scripts/script.sh interfaces
```

### `route`

```bash
scripts/script.sh route
```

### `latency`

```bash
scripts/script.sh latency <host>
```

## Data Storage

Data stored in `~/.local/share/net/`.
