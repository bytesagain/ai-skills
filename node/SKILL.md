# node

Manage Node.js projects with dependency analysis and script running. Use when auditing packages, checking outdated deps, or cleaning node_modules.

## Commands

### `deps`

```bash
scripts/script.sh deps <dir>
```

### `scripts`

```bash
scripts/script.sh scripts <dir>
```

### `outdated`

```bash
scripts/script.sh outdated <dir>
```

### `audit`

```bash
scripts/script.sh audit <dir>
```

### `size`

```bash
scripts/script.sh size <dir>
```

### `clean`

```bash
scripts/script.sh clean <dir>
```

## Data Storage

Data stored in `~/.local/share/node/`.
