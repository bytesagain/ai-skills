# Oh My Bash

A delightful community-driven framework for managing your bash configuration, and an auto-update tool so that makes it easy to keep up with the latest updates from the community.

Inspired by [ohmybash/oh-my-bash]([configured-endpoint]) (7,320+ GitHub stars).

## Commands

- `help` - Help
- `run` - Run
- `info` - Info
- `status` - Status

## Features

- Core functionality from ohmybash/oh-my-bash

## Usage

Run any command: `oh-my-bash <command> [args]`

Returns structured data to stdout. Redirect to a file with `oh-my-bash run > output.txt`.
