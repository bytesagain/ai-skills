# Omni

The all-in-one tool to boost your productivity ⌨️ ## Commands

- `help` - Help
- `run` - Run
- `info` - Info
- `status` - Status

## Features

- Core functionality from alyssaxuu/omni-tool

## Usage

Run any command: `omni-tool <command> [args]`

## Examples

```bash
omni-tool help
omni-tool run
```

## When to Use

- to automate omni tasks in your workflow
- for batch processing tool operations

## Output

Returns structured data to stdout. Redirect to a file with `omni-tool run > output.txt`.

## Configuration

Set `OMNI_TOOL_DIR` environment variable to change the data directory. Default: `~/.local/share/omni-tool/`
