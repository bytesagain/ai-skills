# Omni Tools

Self-hosted collection of powerful web-based tools for everyday tasks. No ads, no tracking, just fast, accessible utilities right from your browser! ## Commands

- `help` - Help
- `run` - Run
- `info` - Info
- `status` - Status

## Features

- Core functionality from iib0011/omni-tools

## Usage

Run any command: `omni-tools <command> [args]`
