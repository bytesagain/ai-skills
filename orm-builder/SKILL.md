# Prisma1

💾 Database Tools incl. ORM, Migrations and Admin UI (Postgres, MySQL & MongoDB) [deprecated] ## Commands

- `help` - Help
- `run` - Run
- `info` - Info
- `status` - Status

## Features

- Core functionality from prisma/orm-builder

## Usage

Run any command: `orm-builder <command> [args]`

## Examples

```bash
orm-builder help
orm-builder run
```

## When to Use

- when you need quick orm builder from the command line
- to automate orm tasks in your workflow

## Output

Returns structured data to stdout. Redirect to a file with `orm-builder run > output.txt`.


## Features

- Simple command-line interface for quick access
- Local data storage with JSON/CSV export
- History tracking and activity logs
- Search across all entries
- No external dependencies required

## Quick Start

```bash
# Check status
orm-builder status

# View help
orm-builder help

# View statistics
orm-builder stats

# Export data
orm-builder export json
```

## How It Works

Orm Builder stores all data locally in `~/.local/share/orm-builder/`. Each command logs activity with timestamps for full traceability.
