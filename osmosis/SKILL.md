# Osmosis — Membrane Separation Technology Reference

Quick-reference skill for osmosis processes, membrane technology, desalination, and water treatment engineering.

## When to Use

- Understanding osmotic pressure and membrane transport
- Designing reverse osmosis desalination systems
- Selecting membrane types for water treatment
- Calculating energy requirements for RO systems
- Troubleshooting membrane fouling and performance decline

## Commands

### `intro`

```bash
scripts/script.sh intro
```

Overview of osmosis — natural osmosis, osmotic pressure, membrane fundamentals.

### `reverse`

```bash
scripts/script.sh reverse
```

Reverse osmosis (RO) — process, system design, pressure requirements, recovery rates.

### `membranes`

```bash
scripts/script.sh membranes
```

Membrane types — TFC, CTA, ceramic, hollow fiber, spiral wound modules.

### `desalination`

```bash
scripts/script.sh desalination
```

Desalination engineering — seawater RO, brackish water, energy recovery devices.

### `fouling`

```bash
scripts/script.sh fouling
```

Membrane fouling — types, causes, prevention, cleaning strategies.

### `pretreatment`

```bash
scripts/script.sh pretreatment
```

Pretreatment systems — sediment removal, antiscalants, pH adjustment, UF prefiltration.

### `energy`

```bash
scripts/script.sh energy
```

Energy analysis — thermodynamic minimum, SEC, energy recovery, renewable integration.

### `forward`

```bash
scripts/script.sh forward
```

Forward osmosis and emerging membrane technologies — PRO, ED, MDC.

### `help`

```bash
scripts/script.sh help
```

### `version`

```bash
scripts/script.sh version
```
