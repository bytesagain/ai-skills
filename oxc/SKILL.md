## Features

- Simple command-line interface for quick access
- Local data storage with JSON/CSV export
- History tracking and activity logs
- Search across all entries
- No external dependencies required

## Quick Start

```bash
# Check status
oxc status

# View help
oxc help

# View statistics
oxc stats

# Export data
oxc export json
```

## How It Works

Oxc stores all data locally in `~/.local/share/oxc/`. Each command logs activity with timestamps for full traceability.
