# Grapesjs

Free and Open source Web Builder Framework. Next generation tool for building templates without coding ## Commands

- `help` - Help
- `run` - Run
- `info` - Info
- `status` - Status

## Features

- Core functionality from GrapesJS/page-builder

## Usage

Run any command: `page-builder <command> [args]`

## Examples

```bash
page-builder help
page-builder run
```

## When to Use

- for batch processing builder operations
- as part of a larger automation pipeline

## Output

Returns summaries to stdout. Redirect to a file with `page-builder run > output.txt`.


## Features

- Simple command-line interface for quick access
- Local data storage with JSON/CSV export
- History tracking and activity logs
- Search across all entries
- No external dependencies required

## Quick Start

```bash
# Check status
page-builder status

# View help
page-builder help

# View statistics
page-builder stats

# Export data
page-builder export json
```

## How It Works

Page Builder stores all data locally in `~/.local/share/page-builder/`. Each command logs activity with timestamps for full traceability.
