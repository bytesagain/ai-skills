# Monolith

⬛️ CLI tool and library for saving complete web pages as a single HTML file ## Commands

- `help` - Help
- `run` - Run
- `info` - Info
- `status` - Status

## Features

- Core functionality from Y2Z/page-saver

## Usage

Run any command: `page-saver <command> [args]`

## Examples

```bash
page-saver help
page-saver run
```

## When to Use

- when you need quick page saver from the command line
- to automate page tasks in your workflow

## Output

Returns results to stdout. Redirect to a file with `page-saver run > output.txt`.


## Features

- Simple command-line interface for quick access
- Local data storage with JSON/CSV export
- History tracking and activity logs
- Search across all entries
- No external dependencies required

## Quick Start

```bash
# Check status
page-saver status

# View help
page-saver help

# View statistics
page-saver stats

# Export data
page-saver export json
```

## How It Works

Page Saver stores all data locally in `~/.local/share/page-saver/`. Each command logs activity with timestamps for full traceability.
