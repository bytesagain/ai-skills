# Phantomjs

Scriptable Headless Browser

Inspired by [ariya/phantomjs]([configured-endpoint]) (29,487+ GitHub stars).

## Commands

- `help` - Help
- `run` - Run
- `info` - Info
- `status` - Status

## Features

- Core functionality from ariya/phantomjs

## Usage

Run any command: `phantomjs <command> [args]`

## Examples

```bash
phantomjs help
phantomjs run
```

## When to Use

- as part of a larger automation pipeline
- when you need quick phantomjs from the command line

## Output

Returns logs to stdout. Redirect to a file with `phantomjs run > output.txt`.

## Configuration

Set `PHANTOMJS_DIR` environment variable to change the data directory. Default: `~/.local/share/phantomjs/`
