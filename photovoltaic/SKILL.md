# Photovoltaic — Solar PV Technology Reference

Quick-reference skill for photovoltaic cell physics, module technology, system design, and performance optimization.

## When to Use

- Understanding solar cell physics and efficiency limits
- Selecting PV module technology (mono, poly, thin-film, bifacial)
- Sizing a PV system for residential or commercial application
- Choosing inverter topology (string, micro, central, hybrid)
- Analyzing PV system performance and degradation

## Commands

### `intro`

```bash
scripts/script.sh intro
```

Overview of photovoltaics — photoelectric effect, cell physics, efficiency limits.

### `cells`

```bash
scripts/script.sh cells
```

Solar cell technologies: monocrystalline, polycrystalline, thin-film, perovskite, tandem.

### `modules`

```bash
scripts/script.sh modules
```

PV module construction — encapsulation, bypass diodes, junction box, half-cut cells.

### `inverters`

```bash
scripts/script.sh inverters
```

Inverter technologies — string, micro, central, hybrid, grid-forming vs grid-following.

### `sizing`

```bash
scripts/script.sh sizing
```

PV system sizing — load analysis, peak sun hours, derating, tilt and azimuth optimization.

### `mounting`

```bash
scripts/script.sh mounting
```

Mounting systems — rooftop, ground-mount, trackers, ballast, structural loading.

### `performance`

```bash
scripts/script.sh performance
```

Performance metrics — PR, CUF, specific yield, degradation, monitoring.

### `economics`

```bash
scripts/script.sh economics
```

PV economics — LCOE, payback period, net metering, feed-in tariffs, PPA.

### `help`

```bash
scripts/script.sh help
```

### `version`

```bash
scripts/script.sh version
```
