# Phpstan

PHP Static Analysis Tool - discover bugs in your code without running it! ## Commands

- `help` - Help
- `run` - Run
- `info` - Info
- `status` - Status

## Features

- Core functionality from php-analyzer/php-analyzer

## Usage

Run any command: `php-analyzer <command> [args]`

## Examples

```bash
php-analyzer help
php-analyzer run
```

## When to Use

- to automate php tasks in your workflow
- for batch processing analyzer operations

## Output

Returns structured data to stdout. Redirect to a file with `php-analyzer run > output.txt`.

## Configuration

Set `PHP_ANALYZER_DIR` environment variable to change the data directory. Default: `~/.local/share/php-analyzer/`
