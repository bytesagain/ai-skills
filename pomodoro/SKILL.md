# Pomodoro

Master your focus with the Pomodoro Technique. Work in focused sprints, track your progress.

## Commands

- `start [task]` — Start a pomodoro session with optional task name
- `today` — View today's completed pomodoros
- `week` — Weekly focus summary with visual tomato counts
- `stats` — Overall productivity statistics and top tasks
- `config [key value]` — View or set timer config (work/short_break/long_break minutes)
- `help` — Show commands

## Usage Examples

```bash
pomodoro start Write project proposal
pomodoro start Code review
pomodoro today
pomodoro week
pomodoro config work 30
```
