# Pop — Stack & Queue Operations Reference

Quick-reference skill for pop/push operations, stack-based algorithms, expression evaluation, and backtracking patterns.

## When to Use

- Understanding stack (LIFO) and queue (FIFO) pop semantics
- Implementing undo/redo systems
- Evaluating expressions using stack-based parsing (postfix, shunting yard)
- Designing backtracking algorithms (DFS, maze solving, permutations)
- Comparing pop behavior across programming languages

## Commands

### `intro`

```bash
scripts/script.sh intro
```

Overview of pop operations — LIFO vs FIFO, stack vs queue fundamentals.

### `languages`

```bash
scripts/script.sh languages
```

Pop behavior across languages — JavaScript, Python, Java, C++, Rust, Go.

### `expression`

```bash
scripts/script.sh expression
```

Expression evaluation — postfix notation, shunting-yard algorithm, AST building.

### `undo`

```bash
scripts/script.sh undo
```

Undo/redo systems — command pattern, memento pattern, operational transform.

### `backtrack`

```bash
scripts/script.sh backtrack
```

Backtracking algorithms — DFS, N-Queens, Sudoku, maze solving via stack.

### `callstack`

```bash
scripts/script.sh callstack
```

Call stack mechanics — function calls, recursion, stack overflow, tail calls.

### `concurrent`

```bash

```

Concurrent stacks and queues — lock-free, CAS, work-stealing deques.

### `applications`


Stack applications — bracket matching, browser history, memory allocation, VM execution.
