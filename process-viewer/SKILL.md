# Systeminformer

A free, powerful, multi-purpose tool that helps you monitor system resources, debug software and detect malware. Brought to you by Winsider Seminars & Solutions, Inc. @ [configured-endpoint] ## Commands

- `help` - Help
- `run` - Run
- `info` - Info
- `status` - Status

## Features

- Core functionality from winsiderss/process-viewer

## Usage

Run any command: `process-viewer <command> [args]`


## When to Use

- to automate process tasks in your workflow
- for batch processing viewer operations

## Output

Returns structured data to stdout. Redirect to a file with `process-viewer run > output.txt`.

## Configuration

Set `PROCESS_VIEWER_DIR` environment variable to change the data directory. Default: `~/.local/share/process-viewer/`
