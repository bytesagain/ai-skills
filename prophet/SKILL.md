# Prophet

Tool for producing high quality forecasts for time series data that has multiple seasonality with linear or non-linear growth.

Inspired by [facebook/prophet]([configured-endpoint]) (20,068+ GitHub stars).

## Commands

- `help` - Help
- `run` - Run
- `info` - Info
- `status` - Status

## Features

- Core functionality from facebook/prophet

## Usage

Run any command: `prophet <command> [args]`
