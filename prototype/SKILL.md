# Prototype

Rapid prototyping toolkit — generate HTML prototypes, interactive mockups, click-through demos, component libraries, and user testing scripts.

## Commands

| Command | Description |
|---------|-------------|
| `prototype create` | <name> |
| `prototype page` | <layout> |
| `prototype link` | <from> <to> |
| `prototype preview` | Preview |
| `prototype export` | <format> |
| `prototype test` | <scenario> |

## Usage

```bash
# Show help
prototype help

# Quick start
prototype create <name>
```

## Examples

```bash
# Example 1
prototype create <name>

# Example 2
prototype page <layout>
```

- Run `prototype help` for all available commands
