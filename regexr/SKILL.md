# Regexr

RegExr是一个基于HTML/JS的强大工具，用于创建、测试和学习正则表达式。它提供了实时匹配、模式构建和详细的学习资源，帮助开发者和非技术用户轻松掌握和应用正则表达式。

## 功能特性 (Features)

-   基于 gskinner/regexr 的核心功能。
-   实时正则表达式匹配和测试。
-   可视化正则表达式结构。
-   提供常用正则表达式模式库。

## 使用方法 (Usage)

您可以通过以下命令运行RegExr工具的各项功能：

-   `regexr help`：获取详细帮助。
-   `regexr run`：启动RegExr主界面。
-   `regexr info`：查看工具信息。
-   `regexr status`：检查运行状态。
