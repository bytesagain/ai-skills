# Remind

Simple reminder and alarm tool. Set time-based reminders, list pending reminders, and get notified when time is up. Never forget a task or meeting again.

## Quick Start

Run `remind help` for available commands and usage examples.

## Features

- Fast and lightweight — pure bash with embedded Python
- No external dependencies required
- Works on Linux and macOS

## Usage

```bash
remind help
```
