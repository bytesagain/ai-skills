# Why Did You Render

render-debugger by Welldone Software monkey patches React to notify you about potentially avoidable re-renders. (Works with React Native as well.) ## Commands

- `help` - Help
- `run` - Run
- `info` - Info
- `status` - Status

## Features

- Core functionality from welldone-software/render-debugger

## Usage

Run any command: `render-debugger <command> [args]`

## Examples

```bash
render-debugger help
render-debugger run
```

## When to Use

- as part of a larger automation pipeline
- when you need quick render debugger from the command line

## Output

Returns structured data to stdout. Redirect to a file with `render-debugger run > output.txt`.

## Configuration

Set `RENDER_DEBUGGER_DIR` environment variable to change the data directory. Default: `~/.local/share/render-debugger/`
