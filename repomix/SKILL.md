# Repomix

📦 Repomix is a powerful tool that packs your entire repository into a single, AI-friendly file. Perfect for when you need to feed your codebase to Large Language Models (LLMs) or other AI tools like Claude, ChatGPT, DeepSeek, Perplexity, Gemini, Gemma, Llama, Grok, and more.

Inspired by [yamadashy/repomix]([configured-endpoint]) (22,430+ GitHub stars).

## Commands

- `help` - Help
- `run` - Run
- `info` - Info
- `status` - Status

## Features

- Core functionality from yamadashy/repomix

## Usage

Run any command: `repomix <command> [args]`

Returns reports to stdout. Redirect to a file with `repomix run > output.txt`.
