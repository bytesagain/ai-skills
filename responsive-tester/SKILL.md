# Responsively App

A modified web browser that helps in responsive web development. A web developer's must have dev-tool. ## Commands

- `help` - Help
- `run` - Run
- `info` - Info
- `status` - Status

## Features

- Core functionality from responsively-org/responsive-tester

## Usage

Run any command: `responsive-tester <command> [args]`
