# Hacker Roadmap

A collection of hacking tools, resources and references to practice ethical hacking. ## Commands

- `help` - Help
- `run` - Run
- `info` - Info
- `status` - Status

## Features

- Core functionality from sundowndev/security-roadmap

## Usage

Run any command: `security-roadmap <command> [args]`

## Output

Returns results to stdout. Redirect to a file with `security-roadmap run > output.txt`.

## Configuration

Set `SECURITY_ROADMAP_DIR` environment variable to change the data directory. Default: `~/.local/share/security-roadmap/`
