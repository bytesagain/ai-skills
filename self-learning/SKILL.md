# Self Learning

- **name**: Self Learning
- **description**: Continuous improvement framework for AI agents. Reflect, evaluate, set goals, journal, and benchmark performance. Commands: reflect, evaluate, goals, track, journal, insights, habits, benchmark. Use for AI self-improvement or tracking agent quality.

## Features

- **Structured Reflection** — Guided prompts for analyzing recent work quality
- **Quality Scoring** — Rate outputs on accuracy, completeness, clarity
- **Goal Setting** — SMART improvement goals with deadlines and metrics
- **Progress Tracking** — Visual progress bars and milestone tracking
- **Daily Journaling** — Record learnings, challenges, and wins
- **Insight Extraction** — Surface patterns from journal entries
- **Habit Building** — Checklists for good agent habits with streaks
- **Benchmarking** — Compare performance metrics across time periods

## Commands

| Command | Purpose |
|---------|---------|
| `reflect` | Self-reflection on work |
| `evaluate` | Score output quality |
| `goals` | Set improvement goals |
| `track` | Track goal progress |
| `journal` | Daily learning journal |
| `insights` | Extract insights |
| `habits` | Good habits checklist |
| `benchmark` | Compare over time |
