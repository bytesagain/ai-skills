# server

Monitor server health with CPU, memory, disk, and process checks. Use when checking system status, finding resource hogs, or monitoring uptime.

## Commands

### `status`

```bash
scripts/script.sh status
```

### `cpu`

```bash
scripts/script.sh cpu
```

### `memory`

```bash
scripts/script.sh memory
```

### `disk`

```bash
scripts/script.sh disk
```

### `processes`

```bash
scripts/script.sh processes <n>
```

### `uptime`

```bash
scripts/script.sh uptime
```

## Data Storage

Data stored in `~/.local/share/server/`.
