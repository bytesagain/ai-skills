# Laf

Laf is a vibrant cloud development platform that provides essential tools like cloud functions, databases, and storage solutions. It enables developers to quickly unleash their creativity and bring practical ideas to life with ease. ## Commands

- `help` - Help
- `run` - Run
- `info` - Info
- `status` - Status

## Features

- Core functionality from labring/serverless-runner

## Usage

Run any command: `serverless-runner <command> [args]`
