# Shell Gpt

A command-line productivity tool powered by AI large language models like GPT-5, will help you accomplish your tasks faster and more efficiently. ## Commands

- `help` - Help
- `run` - Run
- `info` - Info
- `status` - Status

## Features

- Core functionality from TheR1D/shell_gpt

## Usage

Run any command: `shell-ai <command> [args]`

## Examples

```bash
shell-ai help
shell-ai run
```

## When to Use

- as part of a larger automation pipeline
- when you need quick shell ai from the command line

## Output

Returns structured data to stdout. Redirect to a file with `shell-ai run > output.txt`.

## Configuration

Set `SHELL_AI_DIR` environment variable to change the data directory. Default: `~/.local/share/shell-ai/`
