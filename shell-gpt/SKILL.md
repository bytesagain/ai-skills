# Shell Gpt

A command-line productivity tool powered by AI large language models like GPT-5, will help you accomplish your tasks faster and more efficiently.

Inspired by [TheR1D/shell_gpt]([configured-endpoint]) (11,891+ GitHub stars).

## Commands

- `help` - Help
- `run` - Run
- `info` - Info
- `status` - Status

## Features

- Core functionality from TheR1D/shell_gpt

## Usage

Run any command: `shell-gpt <command> [args]`

## Examples

```bash
shell-gpt help
shell-gpt run
```

## When to Use

- to automate shell tasks in your workflow
- for batch processing gpt operations

## Output

Returns reports to stdout. Redirect to a file with `shell-gpt run > output.txt`.

## Configuration

Set `SHELL_GPT_DIR` environment variable to change the data directory. Default: `~/.local/share/shell-gpt/`
