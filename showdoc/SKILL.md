# Showdoc

ShowDoc is a tool greatly applicable for an IT team to share documents online一个非常适合IT团队的在线API文档、技术文档工具

Inspired by [star7th/showdoc]([configured-endpoint]) (12,789+ GitHub stars).

## Commands

- `help` - Help
- `run` - Run
- `info` - Info
- `status` - Status

## Features

- Core functionality from star7th/showdoc

## Usage

Run any command: `showdoc <command> [args]`

## Output

Returns formatted output to stdout. Redirect to a file with `showdoc run > output.txt`.

## Configuration

Set `SHOWDOC_DIR` environment variable to change the data directory. Default: `~/.local/share/showdoc/`
