# Signoz

SigNoz is an open-source observability platform native to OpenTelemetry with logs, traces and metrics in a single application. An open-source alternative to DataDog, NewRelic, etc. 🔥 🖥.   👉  Open source Application Performance Monitoring (APM) & Observability tool

Inspired by [SigNoz/signoz]([configured-endpoint]) (26,079+ GitHub stars).

## Commands

- `help` - Help
- `run` - Run
- `info` - Info
- `status` - Status

## Features

- Core functionality from SigNoz/signoz

## Usage

Run any command: `signoz <command> [args]`

## Examples

```bash
signoz help
signoz run
```

## When to Use

- for batch processing signoz operations

## Output

Returns results to stdout. Redirect to a file with `signoz run > output.txt`.

## Configuration

Set `SIGNOZ_DIR` environment variable to change the data directory. Default: `~/.local/share/signoz/`
