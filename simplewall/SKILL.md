# Simplewall

Simple tool to configure Windows Filtering Platform (WFP) which can configure network activity on your computer.

Inspired by [henrypp/simplewall]([configured-endpoint]) (8,138+ GitHub stars).

## Commands

- `help` - Help
- `run` - Run
- `info` - Info
- `status` - Status

## Features

- Core functionality from henrypp/simplewall
