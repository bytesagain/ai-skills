# Changedetectionio

Best and simplest tool for website change detection, web page monitoring, and website change alerts. Perfect for tracking content changes, price drops, restock alerts, and website defacement monitoring—all for free or enjoy our SaaS plan! ## Commands

- `help` - Help
- `run` - Run
- `info` - Info
- `status` - Status

## Features

- Core functionality from dgtlmoon/changedetection.io

## Usage

Run any command: `site-change-alert <command> [args]`

## Examples

```bash
site-change-alert help
site-change-alert run
```

## When to Use

- as part of a larger automation pipeline
- when you need quick site change alert from the command line

## Output

Returns results to stdout. Redirect to a file with `site-change-alert run > output.txt`.

## Configuration

Set `SITE_CHANGE_ALERT_DIR` environment variable to change the data directory. Default: `~/.local/share/site-change-alert/`
