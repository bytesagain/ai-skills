# Sitemapper
Generate and analyze XML sitemaps. SEO essentials from your terminal.
## Commands
- `generate <urls_file>` — Create sitemap.xml from URL list
- `parse <sitemap_url>` — Parse and list sitemap URLs
- `count <sitemap_url>` — Count URLs in sitemap
- `validate <file>` — Validate sitemap XML format
- `check <sitemap_url>` — Check all sitemap URLs for accessibility
## Usage Examples
```bash
sitemapper generate urls.txt > sitemap.xml
sitemapper parse https://example.com/sitemap.xml
sitemapper count https://example.com/sitemap.xml
```
