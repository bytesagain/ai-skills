# Sniffnet

Comfortably monitor your Internet traffic 🕵️‍♂️

Inspired by [GyulyVGC/sniffnet]([configured-endpoint]) (32,974+ GitHub stars).

## Commands

- `help` - Help
- `run` - Run
- `info` - Info
- `status` - Status

## Features

- Core functionality from GyulyVGC/sniffnet

## Usage

Run any command: `sniffnet <command> [args]`

## Examples

```bash
sniffnet help
sniffnet run
```

## When to Use

- as part of a larger automation pipeline
- when you need quick sniffnet from the command line

## Output

Returns structured data to stdout. Redirect to a file with `sniffnet run > output.txt`.

## Configuration

Set `SNIFFNET_DIR` environment variable to change the data directory. Default: `~/.local/share/sniffnet/`
