# Instapy

📷 Instagram Bot - Tool for automated Instagram interactions ## Commands

- `help` - Help
- `run` - Run
- `info` - Info
- `status` - Status

## Features

- Core functionality from InstaPy/InstaPy

## Usage

Run any command: `social-automator <command> [args]`

## When to Use

- as part of a larger automation pipeline
- when you need quick social automator from the command line

## Output

Returns results to stdout. Redirect to a file with `social-automator run > output.txt`.

## Configuration

Set `SOCIAL_AUTOMATOR_DIR` environment variable to change the data directory. Default: `~/.local/share/social-automator/`
