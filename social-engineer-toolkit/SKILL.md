The Social-Engineer Toolkit (SET) repository from TrustedSec - All new versions of SET will be deployed here.

## Commands

- `help` - Help
- `run` - Run
- `info` - Info
- `status` - Status

## Features

- Core functionality from trustedsec/social-engineer-toolkit

## Usage

Run any command: `social-engineer-toolkit <command> [args]`

## Output

## Configuration

Set `SOCIAL_ENGINEER_TOOLKIT_DIR` environment variable to change the data directory. Default: `~/.local/share/social-engineer-toolkit/`
