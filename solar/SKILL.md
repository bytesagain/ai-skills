# Solar — Solar Energy Reference

Quick-reference skill for solar photovoltaic systems, technology, sizing, and installation.

## When to Use

- Designing a residential or commercial solar PV system
- Comparing panel technologies (mono, poly, thin-film)
- Sizing inverters and battery storage
- Calculating energy yield and payback period
- Understanding net metering and grid-tie configurations

## Commands

### `intro`

```bash
scripts/script.sh intro
```

Overview of solar energy — how PV works, the solar spectrum, and system components.

### `panels`

```bash
scripts/script.sh panels
```

Panel technologies compared: monocrystalline, polycrystalline, thin-film, bifacial.

### `inverters`

```bash
scripts/script.sh inverters
```

Inverter types: string, micro, power optimizers, hybrid — pros, cons, sizing rules.

### `sizing`

```bash
scripts/script.sh sizing
```

System sizing calculations: load analysis, peak sun hours, panel count, wire sizing.

### `mounting`

```bash
scripts/script.sh mounting
```

Mounting systems: roof, ground, trackers — tilt angles, orientation, shading analysis.

### `storage`

```bash
scripts/script.sh storage
```

Battery storage: lithium-ion vs lead-acid, capacity sizing, charge controllers.

### `economics`

```bash
scripts/script.sh economics
```

Financial analysis: LCOE, payback period, IRR, incentives, net metering.

### `maintenance`

```bash
scripts/script.sh maintenance
```

O&M best practices: cleaning, monitoring, degradation rates, common faults.

### `help`

```bash
scripts/script.sh help
```

### `version`

```bash
scripts/script.sh version
```
