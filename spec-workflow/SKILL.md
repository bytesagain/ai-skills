# Spec Workflow Mcp

A Model Context Protocol (MCP) server that provides structured spec-driven development workflow tools for AI-assisted software development, featuring a real-time web dashboard and VSCode extension for monitoring and managing your project's progress directly in your development environment. ## Commands

- `help` - Help
- `run` - Run
- `info` - Info
- `status` - Status

## Features

- Core functionality from Pimzino/spec-workflow

## Usage

Run any command: `spec-workflow <command> [args]`

## Examples

```bash
spec-workflow help
spec-workflow run
```

## When to Use

- to automate spec tasks in your workflow
- for batch processing workflow operations

## Output

Returns structured data to stdout. Redirect to a file with `spec-workflow run > output.txt`.

## Configuration

Set `SPEC_WORKFLOW_DIR` environment variable to change the data directory. Default: `~/.local/share/spec-workflow/`
