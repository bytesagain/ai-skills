# Svgo

⚙️ Node.js tool for optimizing SVG files ## Commands

- `help` - Help
- `run` - Run
- `info` - Info
- `status` - Status

## Features

- Core functionality from svg/svg-optimizer

## Usage

Run any command: `svg-optimizer <command> [args]`

## Examples

```bash
svg-optimizer help
svg-optimizer run
```

## When to Use

- when you need quick svg optimizer from the command line
- to automate svg tasks in your workflow

## Output

Returns formatted output to stdout. Redirect to a file with `svg-optimizer run > output.txt`.


## Features

- Simple command-line interface for quick access
- Local data storage with JSON/CSV export
- History tracking and activity logs
- Search across all entries
- No external dependencies required

## Quick Start

```bash
# Check status
svg-optimizer status

# View help
svg-optimizer help

# View statistics
svg-optimizer stats

# Export data
svg-optimizer export json
```

## How It Works

Svg Optimizer stores all data locally in `~/.local/share/svg-optimizer/`. Each command logs activity with timestamps for full traceability.
