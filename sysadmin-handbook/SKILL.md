# The Book Of Secret Knowledge

A collection of inspiring lists, manuals, cheatsheets, blogs, hacks, one-liners, cli/web tools and more. ## Commands

- `help` - Help
- `run` - Run
- `info` - Info
- `status` - Status

## Features

- Core functionality from trimstray/sysadmin-handbook

## Usage

Run any command: `sysadmin-handbook <command> [args]`

```bash
sysadmin-handbook help
sysadmin-handbook run
```

## When to Use

- to automate sysadmin tasks in your workflow
- for batch processing handbook operations

## Output

Returns reports to stdout. Redirect to a file with `sysadmin-handbook run > output.txt`.

## Configuration

Set `SYSADMIN_HANDBOOK_DIR` environment variable to change the data directory. Default: `~/.local/share/sysadmin-handbook/`
