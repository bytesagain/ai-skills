# Fastfetch

A maintained, feature-rich and performance oriented, neofetch like system information tool. ## Commands

- `help` - Help
- `run` - Run
- `info` - Info
- `status` - Status

## Features

- Core functionality from sysinfo-fetch-cli/sysinfo-fetch

## Usage

Run any command: `sysinfo-fetch <command> [args]`

## Examples

```bash
sysinfo-fetch help
sysinfo-fetch run
```

## When to Use

- when you need quick sysinfo fetch from the command line
- to automate sysinfo tasks in your workflow

## Output

Returns summaries to stdout. Redirect to a file with `sysinfo-fetch run > output.txt`.
