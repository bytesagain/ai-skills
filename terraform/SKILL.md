# Terraform

Terraform enables you to safely and predictably create, change, and improve infrastructure. It is a source-available tool that codifies APIs into declarative configuration files that can be shared amongst team members, treated as code, edited, reviewed, and versioned. ## Commands

- `help` - Help
- `run` - Run
- `info` - Info
- `status` - Status

## Features

- Core functionality from hashicorp/terraform

## Usage

Run any command: `terraform <command> [args]`

## Output

Returns summaries to stdout. Redirect to a file with `terraform run > output.txt`.

## Configuration

Set `TERRAFORM_DIR` environment variable to change the data directory. Default: `~/.local/share/terraform/`
