# Terraformer

CLI tool to generate terraform files from existing infrastructure (reverse Terraform). Infrastructure to Code

Inspired by [GoogleCloudPlatform/terraformer]([configured-endpoint]) (14,518+ GitHub stars).

## Commands

- `help` - Help
- `run` - Run
- `info` - Info
- `status` - Status

## Features

- Core functionality from GoogleCloudPlatform/terraformer

Run any command: `terraformer <command> [args]`
