# Terragrunt

Terragrunt is a flexible orchestration tool that allows Infrastructure as Code written in OpenTofu/Terraform to scale.

Inspired by [gruntwork-io/terragrunt]([configured-endpoint]) (9,394+ GitHub stars).

## Commands

- `help` - Help
- `run` - Run
- `info` - Info
- `status` - Status

## Features

- Core functionality from gruntwork-io/terragrunt

## Usage

Run any command: `terragrunt <command> [args]`
