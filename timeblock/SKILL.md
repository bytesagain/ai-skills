# TimeBlock

Block your time. Own your day. Plan every hour with intention.

## Commands

- `plan <HH:MM> <task>` — Block a time slot for a task
- `today` — View today's time blocks with status
- `done <HH:MM>` — Mark a time block as complete
- `clear` — Clear today's schedule
- `template [type]` — Load a schedule template (workday/creative)
- `help` — Show commands

## Usage Examples

```bash
timeblock plan 09:00 Deep work on project
timeblock plan 10:30 Team standup
timeblock plan 12:00 Lunch break
timeblock today
timeblock done 09:00
timeblock template workday
```
