# todo-manager

待办管理工具。添加任务、优先级排序、艾森豪威尔矩阵、番茄钟计划、周计划、GTD方法。Todo manager with task adding, priority sorting, Eisenhower matrix, Pomodoro technique, weekly planning, GTD methodology.

## 核心特点

🎯 **精准** — 针对具体场景定制化输出
📋 **全面** — 多个命令覆盖完整工作流
🇨🇳 **本土化** — 完全适配中文用户习惯

## 命令列表

| 命令 | 功能 |
|------|------|
| `add` | add |
| `prioritize` | prioritize |
| `eisenhower` | eisenhower |
| `pomodoro` | pomodoro |
| `weekly` | weekly |
| `gtd` | gtd |


## 专业建议

- 紧急+重要** → 立即做
- 重要不紧急** → 计划做
- 紧急不重要** → 委托做
- 不紧急不重要** → 删除它
- 个番茄 = 25分钟专注 + 5分钟休息
