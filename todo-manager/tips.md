# Todo Manager Tips

## 方法论速查

### 艾森豪威尔矩阵
- **紧急+重要** → 立即做
- **重要不紧急** → 计划做
- **紧急不重要** → 委托做
- **不紧急不重要** → 删除它

### 番茄钟技巧
- 1个番茄 = 25分钟专注 + 5分钟休息
- 每4个番茄后休息15-30分钟
- 番茄钟期间不中断，记下干扰项稍后处理

### GTD五步法
1. **收集** — 把所有事情倒出来
2. **处理** — 这是什么？需要行动吗？
3. **组织** — 放到正确的位置
4. **回顾** — 定期检查清单
5. **执行** — 选择并行动

### 优先级设定
- 使用 ABCDE 法：A=必须做, B=应该做, C=可以做, D=委托, E=消除
- 先吃青蛙：每天先做最难的任务

## 使用技巧
1. 任务之间用逗号分隔
2. 优先级用冒号标注（高/中/低 或 h/m/l）
3. 时间估算用数字+单位（如 2h, 30min）
