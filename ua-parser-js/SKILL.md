# Ua Parser Js

UAParser.js - The Essential Web Development Tool for User-Agent Detection. Detect Browsers, OS, Devices, Bots, Apps, AI Crawlers, and more. Run in Browser (client-side) or Node.js (server-side).

Inspired by [faisalman/ua-parser-js]([configured-endpoint]) (10,079+ GitHub stars).

## Commands

- `help` - Help
- `run` - Run
- `info` - Info
- `status` - Status

## Features

- Core functionality from faisalman/ua-parser-js

## Usage

Run any command: `ua-parser-js <command> [args]`

## Output

Returns summaries to stdout. Redirect to a file with `ua-parser-js run > output.txt`.

## Configuration

Set `UA_PARSER_JS_DIR` environment variable to change the data directory. Default: `~/.local/share/ua-parser-js/`
