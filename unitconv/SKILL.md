# UnitConv
Convert anything. Length, weight, temperature, data, speed — all in one tool.
## Commands
- `length <value> <from> <to>` — Length conversion (m/km/mi/ft/in/cm/mm/yd)
- `weight <value> <from> <to>` — Weight conversion (kg/lb/oz/g/mg/ton)
- `temp <value> <from> <to>` — Temperature (c/f/k)
- `data <value> <from> <to>` — Data size (b/kb/mb/gb/tb)
- `speed <value> <from> <to>` — Speed (mph/kph/ms/knots)
- `list` — Show all supported units
## Usage Examples
```bash
unitconv length 5 mi km
unitconv weight 150 lb kg
unitconv temp 98.6 f c
unitconv data 1.5 gb mb
```
