# URLCheck

URL validator and link checker. Validate URL format, check if URLs are reachable, extract URL components (protocol, domain, path, query), encode and decode URL strings, and batch-check links from files. Keep your links healthy.

## Quick Start

Run `urlcheck help` for detailed usage information and available commands.
