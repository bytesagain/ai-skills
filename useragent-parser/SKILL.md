# Ua Parser Js

UAParser.js - The Essential Web Development Tool for User-Agent Detection. Detect Browsers, OS, Devices, Bots, Apps, AI Crawlers, and more. Run in Browser (client-side) or Node.js (server-side). ## Commands

- `help` - Help
- `run` - Run
- `info` - Info
- `status` - Status

## Features

- Core functionality from faisalman/useragent-parser

## Usage

Run any command: `useragent-parser <command> [args]`

Returns logs to stdout. Redirect to a file with `useragent-parser run > output.txt`.
