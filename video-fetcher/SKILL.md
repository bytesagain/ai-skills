# Lux

👾 Fast and simple video download library and CLI tool written in Go ## Commands

- `help` - Help
- `run` - Run
- `info` - Info
- `status` - Status

## Features

- Core functionality from iawia002/video-fetcher

## Usage

Run any command: `video-fetcher <command> [args]`

## Examples

```bash
video-fetcher help
video-fetcher run
```

## When to Use

- to automate video tasks in your workflow
- for batch processing fetcher operations

## Output

Returns summaries to stdout. Redirect to a file with `video-fetcher run > output.txt`.

## Configuration

Set `VIDEO_FETCHER_DIR` environment variable to change the data directory. Default: `~/.local/share/video-fetcher/`
