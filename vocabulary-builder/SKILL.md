# vocabulary-builder

词汇学习与记忆工具。中英词汇学习、语境例句、词根词缀分析、词汇测试、学习计划、分级词汇表。Vocabulary builder with contextual examples, etymology, word roots, tests, study plans, graded word lists.

## 核心特点

🎯 **精准** — 针对具体场景定制化输出
📋 **全面** — 多个命令覆盖完整工作流
🇨🇳 **本土化** — 完全适配中文用户习惯

## 可用命令

- **learn** — learn
- **context** — context
- **root** — root
- **test** — test
- **A** — A
- **B** — B
- **C** — C
- **D** — D


## 专业建议

- 学习新词**：输入单词获取完整释义、音标、词性、例句
- 语境例句**：在不同场景下展示单词用法
- 词根词缀**：
- 前缀：un-, dis-, pre-, re-, anti-
- 后缀：-tion, -ment, -able, -ful
