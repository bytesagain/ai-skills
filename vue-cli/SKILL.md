# Vue Cli

🛠️ webpack-based tooling for Vue.js Development

Inspired by [vuejs/vue-cli]([configured-endpoint]) (29,636+ GitHub stars).

## Commands

- `help` - Help
- `run` - Run
- `info` - Info
- `status` - Status

## Features

- Core functionality from vuejs/vue-cli

## Usage

Run any command: `vue-cli <command> [args]`

## Examples

```bash
vue-cli help
vue-cli run
```

## When to Use

- as part of a larger automation pipeline
- when you need quick vue cli from the command line

## Output

Returns logs to stdout. Redirect to a file with `vue-cli run > output.txt`.

## Configuration

Set `VUE_CLI_DIR` environment variable to change the data directory. Default: `~/.local/share/vue-cli/`
