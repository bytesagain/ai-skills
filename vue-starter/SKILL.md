# Vue Cli

🛠️ webpack-based tooling for Vue.js Development ## Commands

- `help` - Help
- `run` - Run
- `info` - Info
- `status` - Status

## Features

- Core functionality from vuejs/vue-starter

## Usage

Run any command: `vue-starter <command> [args]`

## Examples

```bash
vue-starter help
vue-starter run
```

## When to Use

- as part of a larger automation pipeline
- when you need quick vue starter from the command line

## Output

Returns summaries to stdout. Redirect to a file with `vue-starter run > output.txt`.

## Configuration

Set `VUE_STARTER_DIR` environment variable to change the data directory. Default: `~/.local/share/vue-starter/`


## Features

- Simple command-line interface for quick access
- Local data storage with JSON/CSV export
- History tracking and activity logs
- Search across all entries
- No external dependencies required

## Quick Start

```bash
# Check status
vue-starter status

# View help
vue-starter help

# View statistics
vue-starter stats

# Export data
vue-starter export json
```

## How It Works

Vue Starter stores all data locally in `~/.local/share/vue-starter/`. Each command logs activity with timestamps for full traceability.
