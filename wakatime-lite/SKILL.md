# WakaTime Lite
Track your coding time. See where your hours go. Set goals. Stay productive.
## Commands
- `start <project>` — Start tracking a project
- `stop` — Stop current tracking
- `status` — Show current tracking status
- `today` — Today's coding time summary
- `week` — Weekly summary
- `projects` — Time by project
- `goal <minutes>` — Set daily coding goal
## Usage Examples
```bash
wakatime-lite start my-project
wakatime-lite status
wakatime-lite stop
wakatime-lite today
wakatime-lite week
```
