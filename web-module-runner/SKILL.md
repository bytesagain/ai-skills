# Wmr

👩‍🚀 The tiny all-in-one development tool for modern web apps. ## Commands

- `help` - Help
- `run` - Run
- `info` - Info
- `status` - Status

## Features

- Core functionality from preactjs/web-module-runner

## Usage

Run any command: `web-module-runner <command> [args]`

## Examples

```bash
web-module-runner help
web-module-runner run
```

## When to Use

- when you need quick web module runner from the command line
- to automate web tasks in your workflow

## Output

Returns logs to stdout. Redirect to a file with `web-module-runner run > output.txt`.


## Features

- Simple command-line interface for quick access
- Local data storage with JSON/CSV export
- History tracking and activity logs
- Search across all entries
- No external dependencies required

## Quick Start

```bash
# Check status
web-module-runner status

# View help
web-module-runner help

# View statistics
web-module-runner stats

# Export data
web-module-runner export json
```

## How It Works

Web Module Runner stores all data locally in `~/.local/share/web-module-runner/`. Each command logs activity with timestamps for full traceability.
