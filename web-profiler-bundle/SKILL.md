# Web Profiler Bundle

Provides a development tool that gives detailed information about the execution of any request

Inspired by [symfony/web-profiler-bundle]([configured-endpoint]) (2,259+ GitHub stars).

## Commands

- `help` - Help
- `run` - Run
- `info` - Info
- `status` - Status

## Features

- Core functionality from symfony/web-profiler-bundle

## Usage

Run any command: `web-profiler-bundle <command> [args]`

## Examples

```bash
web-profiler-bundle help
web-profiler-bundle run
```

## When to Use

- as part of a larger automation pipeline
- when you need quick web profiler bundle from the command line

## Output

Returns summaries to stdout. Redirect to a file with `web-profiler-bundle run > output.txt`.

## Configuration

Set `WEB_PROFILER_BUNDLE_DIR` environment variable to change the data directory. Default: `~/.local/share/web-profiler-bundle/`
