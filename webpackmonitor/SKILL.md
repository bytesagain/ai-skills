# Webpackmonitor

A tool for monitoring webpack optimization metrics through the development process

Inspired by [webpackmonitor/webpackmonitor]([configured-endpoint]) (2,402+ GitHub stars).

## Commands

- `help` - Help
- `run` - Run
- `info` - Info
- `status` - Status

## Features

- Core functionality from webpackmonitor/webpackmonitor

## Usage

Run any command: `webpackmonitor <command> [args]`

## Examples

```bash
webpackmonitor help
webpackmonitor run
```

## When to Use

- to automate webpackmonitor tasks in your workflow
- for batch processing webpackmonitor operations

## Output

Returns formatted output to stdout. Redirect to a file with `webpackmonitor run > output.txt`.

## Configuration

Set `WEBPACKMONITOR_DIR` environment variable to change the data directory. Default: `~/.local/share/webpackmonitor/`
