# Wmr

👩‍🚀 The tiny all-in-one development tool for modern web apps.

Inspired by [preactjs/wmr]([configured-endpoint]) (4,932+ GitHub stars).

## Commands

- `help` - Help
- `run` - Run
- `info` - Info
- `status` - Status

## Features

- Core functionality from preactjs/wmr

## Usage

Run any command: `wmr <command> [args]`

- to automate wmr tasks in your workflow

## Output

Returns summaries to stdout. Redirect to a file with `wmr run > output.txt`.
