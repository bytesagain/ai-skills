# WordCraft

Essential text tools for writers. Analyze, outline, and craft better content from your terminal.

## Why WordCraft?

- **Word count**: Detailed stats including reading time
- **Readability scoring**: Flesch-Kincaid grade level analysis
- **Outline generator**: Structured templates for any topic
- **Lorem ipsum**: Quick placeholder text generation
- **Title case**: Instant text formatting

## Commands

- `wordcount <file>` — Count words, characters, lines, sentences, and estimate reading time
- `readability <file>` — Analyze readability with Flesch-Kincaid grade level
- `outline <topic>` — Generate a structured article outline
- `lorem [paragraphs]` — Generate lorem ipsum placeholder text
- `titlecase <text>` — Convert text to title case
- `info` — Version info
- `help` — Show commands

## Usage Examples

```bash
wordcraft wordcount my-article.md
wordcraft readability blog-post.txt
wordcraft outline "The Future of Remote Work"
wordcraft lorem 5
wordcraft titlecase "this is a great headline"
```
