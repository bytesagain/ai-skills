# Deer Flow

An open-source SuperAgent use that researches, codes, and creates. With the help of sandboxes, memories, tools, skills and subagents, it handles different levels of tasks that could take minutes to hours. ## Commands

- `help` - Help
- `run` - Run
- `info` - Info
- `status` - Status

## Features

- Core functionality from bytedance/workflow-engine

## Usage

Run any command: `workflow-engine <command> [args]`

## Examples

```bash
workflow-engine help
workflow-engine run
```

## When to Use

- as part of a larger automation pipeline
- when you need quick workflow engine from the command line

## Output

Returns reports to stdout. Redirect to a file with `workflow-engine run > output.txt`.

## Configuration

Set `WORKFLOW_ENGINE_DIR` environment variable to change the data directory. Default: `~/.local/share/workflow-engine/`
