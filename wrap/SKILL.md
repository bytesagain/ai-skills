# Wrap — Text & Data Wrapping Reference

Quick-reference skill for wrapping — text line breaking, word wrap algorithms, and data wrapping patterns.

## When to Use

- Implementing word wrap for terminal or fixed-width output
- Understanding CSS word-wrap, overflow-wrap, and hyphens
- Using the Knuth-Plass optimal line-breaking algorithm
- Wrapping values in monadic containers (Option, Result)
- Implementing decorator/wrapper patterns in code
