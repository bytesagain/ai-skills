# write

Draft, proofread, and analyze text with readability scoring. Use when writing drafts, checking word counts, or measuring readability.

## Commands

### `wordcount`

```bash
scripts/script.sh wordcount <file>
```

### `readability`

```bash
scripts/script.sh readability <file>
```

### `outline`

```bash
scripts/script.sh outline <topic>
```

### `draft`

```bash
scripts/script.sh draft <topic words>
```

### `proofread`

```bash
scripts/script.sh proofread <file>
```

### `export`

```bash
scripts/script.sh export <file format>
```

## Requirements

- bash 4.0+

## Data Storage

Data stored in `~/.local/share/write/`.
