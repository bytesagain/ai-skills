# Yellowbrick

Visual analysis and diagnostic tools to help machine learning model selection.

Inspired by [DistrictDataLabs/yellowbrick]([configured-endpoint]) (4,396+ GitHub stars).

## Commands

- `help` - Help
- `run` - Run
- `info` - Info
- `status` - Status

## Features

- Core functionality from DistrictDataLabs/yellowbrick

## Usage

Run any command: `yellowbrick <command> [args]`
