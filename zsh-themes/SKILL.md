# Ohmyzsh

🙃 A delightful community-driven (with 2,400+ contributors) framework for managing your zsh configuration. Includes 300+ optional plugins (rails, git, macOS, hub, docker, homebrew, node, php, python, etc), 140+ themes to spice up your morning, and an auto-update tool that makes it easy to keep up with the latest updates from the community. ## Commands

- `help` - Help
- `run` - Run
- `info` - Info
- `status` - Status

## Features

- Core functionality from zsh-themes/zsh-themes

## Usage

Run any command: `zsh-themes <command> [args]`

## Examples

```bash
zsh-themes help
zsh-themes run
```

## When to Use

- to automate zsh tasks in your workflow
- for batch processing themes operations

## Output

Returns results to stdout. Redirect to a file with `zsh-themes run > output.txt`.

## Configuration

Set `ZSH_THEMES_DIR` environment variable to change the data directory. Default: `~/.local/share/zsh-themes/`
